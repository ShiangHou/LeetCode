## 初步思路 
首先就没太看懂题目，什么叫做只能按照横向或者纵向来去划分。后来想到就是，在这个正方形中竖着画一条线，使其两边的价值差最小，那么同理，我也可以只横着画一条线，让两块区域的面积差最小

那么自然就想到前缀和，我只要竖着统计一次，就可以计算横着切的最小的了

假设是
1 2 3  -->6
2 1 3  -->6
1 2 3  -->6

统计横着的,就像上边那样，此时我算横着切的 6 / 12     12/ 6

这里有一个数学小技巧，A+B = total,我们要统计的是A-B的最小值，那么带入就是A-（total-A），那么就是2A-total的绝对值的最小值，即 total-2A的最小值，因此算一个total，然后每次算前缀的时候用total-2A就行

```python 

import sys
input_data = sys.stdin.read()
data = input_data.split()
index = 0
n = int(data[index])
index += 1
m = int(data[index])
index += 1


nums = []
#这里算一个总面积，后面备用
total = 0
for i in range(n):
    temp = []
    for i in range(m):
        temp.append(int(data[index]))
        total += int(data[index])
        index += 1
    nums.append(temp)

#计算横着的和
hor = [0]*n
for i in range(n):
    for j in range(m):
        hor[i] += nums[i][j]
#计算竖着的和
ver = [0]*m
for j in range(m):
    for i in range(n):
        ver[j] += nums[i][j]
#计算划分
ans = float('inf')


#先算横着切的，横着切要用hor
hor_cut = 0
for i in range(n):
    hor_cut += hor[i]
    ans = min(ans,abs(total-2*hor_cut))

#再算竖着的
ver_cut = 0
for j in range(m):
    ver_cut += ver[j]
    ans = min(ans,abs(total-2*ver_cut))

print(ans)




```
