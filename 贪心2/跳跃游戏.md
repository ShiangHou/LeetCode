# 跳跃游戏

力扣55

## 思路

给定一个数组，数组的下标代表了可以跳跃的长度，问能不能从起点跳到终点

这题其实就是在看前面的数组

比如从i位置开始，能不能**覆盖**到最后面，如果能覆盖到后面，那就可以，如果不能，就不行

梳理一下思路，我们需要首先维护一个temp_max，即最远能抵达的长度

然后我们用一个指针去遍历数组，比如到了i位置，那么就是nums[i]加上，然后max（temp_max,nums[i]）取最大的就行，如果最后temp_max还没有数组长度大，那么就说明不是

我们可以用一个while i< temp_max来取弄，即然后遇到更大的就更新temp_max，然后一直推i

```python 
    def canJump(self, nums: List[int]) -> bool:
        if len(nums) == 1:
            return True
        temp_max = nums[0]
        i = 1
        while i <= temp_max :#到了最大的地方也可也跳
            temp_max = max(i+nums[i],temp_max)
            if temp_max >= len(nums)-1:
                return True
            i += 1
        return False


```
