# 替换数字
卡码网
## 初步思路
题目意思是，如果一个字符串，比如a1b2，那么就把其中的是数字的转换为number，（仔细读了一下题目，是每个数字）

初步的思路还是very的清晰，我可以用ASCII来去判断，首先将字符串转为字符串list，如果说是这个格子的ASCII在48-58（感觉不用考虑），那就直接换成“number",最后把字符串拼在一起就行了

```python 
s = str(input())
s_list = list(s)
for i in range(len(s_list)):
    if 48<=ord(s_list[i])<=58:
        s_list[i] = "number"
print("".join(s_list))

```
