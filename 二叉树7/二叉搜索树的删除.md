## 初步思路

首先感觉也是very的简单，依旧是按照性质遍历，如果找到了直接斩杀

斩杀的时候看一下，如果左边有，右边也有，那就让左边合并到右边去，然后接上就行

如果一边有，一边没有，那么也是合并

感觉要新写一个合并的代码啊

感觉有点复杂啊，即如果很深的化会复杂

想了一下其实不复杂，比如左边和右边，那么我只需要把左边的全部加入到**右边的最左端段最底**就行

为什么可以，因为一颗二叉搜索树，右边所有子树一定都是大于左边的，所以左边的过来后就是路边一条，只能待在最底下


```python 
    def deleteNode(self, root: Optional[TreeNode], key: int) -> Optional[TreeNode]:
        if not root:
            return root
        #找到了
        if root.val == key:
            if not root.right:#如果右边没有，直接返回左边的，递归回去让左边的接上
                return root.left
            else:#右边有，那么要把左子树接到右边，然后返回右边
                return self.LeftToRight(root.left,root.right)
        #递归的进行左，递归的进行右
        if root.val < key:#往右走
            root.right = self.deleteNode(root.right,key)
        else:#往左走
            root.left = self.deleteNode(root.left,key)
        return root
            
    
    def LeftToRight(self,left,right):
        if left == None:#左是空的话，不需要合并,直接返回右就行
            return right
        #右是不是空最好在主流程解决
        node = right
        while node:
            if node.left == None:
                node.left = left
                break
            node = node.left
        return right
```
