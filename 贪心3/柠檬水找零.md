# 柠檬水找零

力扣860

题目说，我们再卖柠檬水，钱的面值有 5 10 20 的，一杯水5块

我们最开始的时候没有钱，然后问如果能找零，就返回true，如果不能找0就返回false

这题听起来very的简单啊，首先准备一个dict存放面值和数量，初始为0

然后分情况讨论，如果5元，就收下就行

如果是10元，收下10元，看一下dict有没有5的， 没有5的就不行呗

同理，20的要返回一张5的和一张10的

哎哎等等，20的话，我可以找零3张5元的或者1张5和一张10，如果都有呢？

思考一下，由于5元的比较万金油，还是优先一张5一张10，没有的话再3张5

```python 
    def lemonadeChange(self, bills: List[int]) -> bool:
        cash = {5:0,10:0,20:0}
        for money in bills:
            if money == 5:
                cash[5] += 1
            if money == 10:
                cash[10] += 1
                if cash[5] == 0:
                    return False
                else:
                    cash[5] -= 1
            if money == 20:
                cash[20] += 1
                #优先一张10一张五
                if cash[10] != 0 and cash[5] != 0:
                    cash[10] -= 1
                    cash[5] -= 1
                #不行的话，看有没有三张五
                elif cash[5] >= 3:
                    cash[5] -=3
                else:
                    return False
        return True
```
