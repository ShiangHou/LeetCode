# 反转字符串中的单词

## 初步思路
题目复述一遍，说是对于一个字符串，比如 the sky is blue,将其反转成 blue is sky the,以空格为界限就行

第一个首先想到的就是，我将这个字符串按照空格拆开，打到一个列表里，比如

["the", "sky", "is", "blue"]

然后问题就very的简单了，直接以列表元素为单元去调换，最后再用join拼起来就行

拷打了一下好厚米，好厚米和我说对一个字符串用.split方法可以变成一个list，并且是默认按照空格切分的
```python 
def reverseWords(self, s: str) -> str:
    s_list = s.split()
    l,r = 0,len(s_list)-1
    while l <r:
        s_list[l],s_list[r] = s_list[r],s_list[l]
        l+=1
        r-=1
    return " ".join(s_list)

```
