比如dp[i][j],就是说以第一个数组的i下标和第2个数组的j下标为数组结尾的，最长子序列

这样的话，如果nums1[i]和nums2[j]如果说是不一样的，说明“以这俩结尾的子数组”就不是公共的，那么直接弄成0就行

如果是一样的，那么我们直接让其为dp[i-1][j-1]+1就行

这样有点问题， 因为还是要构建dp n+1和m+1，然后用那个i-1和j-1，因为是前i个和前j个，索引从0开始，因为前i个在nums是i-1

```python 
class Solution:
    def findLength(self, nums1: List[int], nums2: List[int]) -> int:
        m = len(nums1)
        n = len(nums2)
        dp = [[0]*(n+1) for _ in range(m+1)]
        ans = 0
        for i in range(1,m+1):
            for j in range(1,n+1):
                if nums1[i-1]!=nums2[j-1]:
                    dp[i][j] = 0
                else:
                    dp[i][j] = dp[i-1][j-1]+1
                ans = max(ans,dp[i][j])
        return ans 

```
