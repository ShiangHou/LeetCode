# 监控二叉树

力扣968

## 初步思路

题目说，一个二叉树，然后如果我放一个监视器的话，可以监视父节点、左右子节点

给我一个二叉树，问最少放多少个监视器可以监视完整个二叉树

一点思路没有

## 思路

读了一遍厚米的思路，是这样的


1:首先，这个需要从底到顶来去后序遍历

为什么

因为，如果是根节点，那么在根节点上放监视器是很没有效率的，因为它只监视了一个，所以对于根节点，最好的方式是在父节点

所以我们要用后序，左右中，因为这是从底到顶的算法啊，然后不断往上递归就行

2:定义三种状态

我们需要准备三种状态，0 1 2
0状态 无覆盖
1状态 装了摄像头
2状态 有覆盖

这个比较自然

对于none节点，我们可以直接返回2状态，
为什么，因为如果是0，那么就需要在子节点上安摄像头，1状态显然不行，因为会误报子节点，所以是2状态，也就是不管他

3:遍历中的操作
左右中，如果返回得到的是0，那么当前节点就打上1，
如果返回的是1，那么自己打上2再返回
如果返回的是2，那么自己打上0


最后递归到根节点，如果返回0，在根节点打上1就行

所有打上1的时候记得最后的count+1

## 最终思路

忘记考虑左右子树了，
如果左右至少有一个是0，那么当前必须要打上1

如果左右都是2，那么自己返回0。（自己没有被覆盖）

如果左右至少有一个是1，那么自己返回0（自己已经被覆盖了）


```python 
class Solution:
    def minCameraCover(self, root: Optional[TreeNode]) -> int:
        self.count = 0
        #0是未覆盖，1是有监视，2是已覆盖
        if self.f(root) == 0:
            self.count += 1
        return self.count
    
    def f(self,node):
        if not node:
            return 2
        left_state = self.f(node.left)
        right_state = self.f(node.right)
        #如果左右有一个是未覆盖，自己要是1才行
        if left_state == 0 or right_state == 0:
            self.count += 1
            return 1
        #如果左右都是2，那么自己是0
        if left_state == 2 and right_state == 2:
            return 0
        #如果左右有一个是1，那么自己就是2
        if left_state == 1 or right_state == 1:
            return 2


```
