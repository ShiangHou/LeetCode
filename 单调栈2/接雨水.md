# 接雨水
力扣42

## 栈思路
经典题目，

思路是，当我们遍历到i的时候，我们去看i头顶上能放几个水

头顶上能放几个水和左右的最大有关，假设左边数组的最大是10，右边是20，那么只有左边能兜住，

找到小的那个兜住的，然后和自己的高度相减就行，注意还要和0比，因为如果i位置也是很高的话，那么就没有

那么问题转化为，怎么找一个i左边所有的最大，以及右边所有的最大

我们用一个单调栈

比如找左边最大的，那么我们从0开始往后遍历，

遍历到i的时候，如果i它比-1位置的大，那么直接就是他自己，如果没有-1位置的大，直接继承-1位置

例如  4 2 0 3 2 5

4 4 4 4 4 5
然后2 进来发现没4大，那么2左边的最大还是4
同理，2 0 3 2的位置上都是4

如果我们想找0位置右边的最大的，直接取这个数组就行

右边最大的同理，反着遍历一遍就行

```python 
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        left_max = [0]*n
        right_max = [0]*n
        left_max[0] = height[0]
        right_max[-1] = height[-1]
        #X先构建左的单调栈
        for i in range(1,n):
            left_max[i] = max(left_max[i-1],height[i])
        for i in range(n-2,-1,-1):
            right_max[i] = max(right_max[i+1],height[i])
        
        ans = 0
        #然后是主流程
        for i in range(n):
            ans +=  max( min(left_max[i],right_max[i])-height[i],0)  
        return ans 

```
