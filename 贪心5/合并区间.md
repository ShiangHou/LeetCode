# 合并区间

力扣56
## 初步思路

这题就是说，也是一堆二维的数组，他们之间互相可能会有重叠的，然后让我们把重叠的合并就行

思路跟之前的那个数组和气球差不多，先排序，然后看，如果有重叠的，合并，并且吧合并的整理成新的i+1，如果没有，就把i保存就行（i+1还是要继续比较）

记得先排序

```python 
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        intervals.sort(key = lambda x: x[0])
        ans = []
        n = len(intervals)
        for i in range(n-1):
            if intervals[i][1] >= intervals[i+1][0]:
                #开始合并，直接并到i+1就行
                intervals[i+1][0] = intervals[i][0]
                intervals[i+1][1] = max(intervals[i+1][1],intervals[i][1])
            else:
                ans.append(intervals[i])
        ans.append(intervals[-1])#加上最后一个，因为i遍历不到最后一个
        return ans
```
