## 初步思路

题目问，给了一个非负整数，然后可以在这个非负整数前面加一个+或者-，然后求和，最后如果是target，就记录一次组合，问有多少种不同的组合

换句话说，数组中的每个数字都有两个状态，一个+一个-，那么先想一个暴力递归，对于每一个数组，我都有+和-，那么我可以递归到所有的状态组合，然后找到最后等于的就行

```python 
class Solution:
    def findTargetSumWays(self, nums: List[int], target: int) -> int:
        self.count = 0
        self.f(nums,0,target)
        return self.count

    

    def f(self,nums,i,target):
        if i == len(nums):#到终点且搜集到一种
            if sum(nums) == target:
                self.count+= 1
            
            return 
        #将i换成两种状态
        #正的状态，不用管，直接递归就行，本来就是正的
        self.f(nums,i+1,target)
        #负状态
        nums[i] = -nums[i]
        self.f(nums,i+1,target)
        #返回过来
        nums[i] = -nums[i]
```

这个可以过，但是依旧会报错

考虑一下剪枝，首先是最后的求和，因为每次都会计算一下,可以用个cur_sum,即cur_sum每次加一下减一下，这样就行，原来是N2^N,可以变成2^N

```python 
class Solution:
    def findTargetSumWays(self, nums: List[int], target: int) -> int:
        self.count = 0
        self.f(nums,0,0,target)
        return self.count

    

    def f(self,nums,i,cur_sum,target):
        if i == len(nums):#到终点且搜集到一种
            if cur_sum == target:
                self.count+= 1
            
            return 
        #将i换成两种状态
        #正的状态，不用管，直接递归就行，本来就是正的
        self.f(nums,i+1,cur_sum+nums[i],target)
        #负状态
        # nums[i] = -nums[i]
        self.f(nums,i+1,cur_sum-nums[i],target)

```

但是依旧是2^N的复杂度，

## 背包的思路

数学技巧，假设我们现在是正数的是N，是负的是P

假设数组 nums 中所有的元素总和为 $S$。在我们需要求出的某一种合法方案中：设被分配了正号（+）的数字之和为 $P$。设被分配了负号（-）的数字绝对值之和为 $N$。根据题意和基本的数学常识，我们可以列出两个绝对成立的等式：题目的要求： 所有的正数之和减去所有的负数之和，必须等于目标和。即：$P - N = target$客观的物理事实： 无论怎么分配正负号，所有数字的绝对值总和是不变的，等于原数组的总和。即：$P + N = S$2. 解方程提取目标变量我们将上述两个等式进行相加：$(P - N) + (P + N) = target + S$化简得到：$2P = target + S$进而推导出取正号的数字总和 $P$ 的值：$$P = \frac{target + S}{2}$$

问题转为：请在数组 nums 中挑选出一个子集，使得这个子集内元素的总和，恰好等于固定容量 $P$。请问有几种挑选方案？

这样就行了

```python 
class Solution:
    def findTargetSumWays(self, nums: List[int], target: int) -> int:
        if abs(target) > sum(nums) or (sum(nums)+target)%2 != 0:
            return 0
        
        n = len(nums)
        t = (sum(nums)+target) // 2
        dp = [[0]*(t+1) for _ in range(n+1)]#用前i个，然后凑成j的种类有多少，最后返回dp[n][target]就行
        # for i in range(n+1):#凑成0，因为都是非负整数，所以不可能，所以都是0
        #     dp[i][0] = 0
        # for j in range(t+1):#用前0个凑，所以也是0
        #     dp[0][j] = 0

        #那就不用初始化

        #但是从0中取0可以，有一种，就是空集
        dp[0][0] = 1
        for i in range(1,n+1):
            for j in range(t+1):#题目说target可以是0，所以这里的背包重量也可以是0
                #这里就是，之前的去了负的量+取量正的量
                #嗷嗷不是，因为已经转为了背包问题，所以不需要都考虑正负 了，只看取和不取就行
                if j-nums[i-1] >= 0:
                    dp[i][j] = dp[i-1][j] + dp[i-1][j-nums[i-1]]
                else:
                    dp[i][j] = dp[i-1][j]
        return dp[n][t]
```

一维度

所谓的一维度就是我用一个一维在原地修改数组

为了防止这个if条件，我们可以从后往前遍历那个j，然后拿着减就行

```python 
class Solution:
    def findTargetSumWays(self, nums: List[int], target: int) -> int:
        if abs(target) > sum(nums) or (sum(nums)+target)%2 != 0:
            return 0
        t = (sum(nums)+target) // 2
        dp = [0]*(t+1)
        dp[0] = 1
        for num in nums:
            for j in range(t,num-1,-1):#后往前遍历，不用if判断，遍历到num（包含num就是0，即仅有i-1的状态）
                dp[j] += dp[j-num]
        return dp[t]



```
