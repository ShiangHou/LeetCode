## 初步思路
题目让我们找到所有1-n中 所有k个组合的数 比如找1 2 3 4中2个的组合，就是 1 2， 1 3， 1 4， 2 3， 2 4， 3 4

就是所谓的组合数，也就是去重的

可以在脑海中想象一个二叉树从1开始分叉，分为要1和不要1，然后又分别往下，分为要2和不要2，以此类推，那么就是2的n次方个节点

然后每次要的时候收集进入一下，每次回溯的时候吧之前加入的给斩杀掉，

但这样是召回了所有的组合，现在要求说是k个长度，所以只需要筛选就行，准备一个k，如果k满了就停止收集就行

此外，不用担心收集不到下面的，因为还有 1不要，2不要 等等的路径可以谈入到下面的

```python 
    def combine(self, n: int, k: int) -> List[List[int]]:
        self.ans = []
        self.max_n = n
        self.f(1,k,[])
        return self.ans

    def f(self,n,k,path):#n是当前遍历到的数字，选择要还是不要，k是还需要搜集的数量，path是当前搜集到的路径
        if k == 0:
            self.ans.append(path.copy())#每次注意保存的是浅拷贝的，要不然全是0
            return 
        if n > self.max_n:#遇到大的就直接停止
            return 
        #要n
        path.append(n)
        #然后往下走
        self.f(n+1,k-1,path)
        #回来了，先把n去掉
        path.pop()
        #然后走没有n的
        self.f(n+1,k,path)
        return path

```
