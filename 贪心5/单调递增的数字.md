## 初步思路

题目说，给我们一个数，看一下这个数是不是单调递增的（但其实根据题意是单调不减的），然后找到小于等于这个数的最大的单调递增的数

这题首先从后往前遍历暴力解，应该是不可行

思路应该是，先看个位，如果个位比前一位要小，那么前一位就要-1，然后自己变成9，然后再拿前一位和前前一位去比较就行

比如说 32 ，因为2比3大，所以2变成9，3变成3-1 = 2，最后就是29

332，以此类推，变成329，然后2比3大，2变9，然后3变3-1，最后就是299

嘶，但是有个问题，比如说 3324，最后会变成2994，是不对的，所以说，需要看一下i-1位置的和i位置的取一个最小的就行



```python 
class Solution:
    def monotoneIncreasingDigits(self, n: int) -> int:
        n_ = str(n)
        n_str = [i for i in n_]#先变成列表，字符串是不能改的
        for i in range(len(n_str)-1,0,-1):#从个位开始
            #如果是最后一个，那么就不用比i+1的
            if i== len(n_str)-1:
                if int(n_str[i])< int(n_str[i-1]):#如果后比前小了
                    n_str[i] = str(9)
                    n_str[i-1] = str(int(n_str[i-1])-1)#前面的要-1
            #正常不是最后一个的
            if int(n_str[i])< int(n_str[i-1]):#如果后比前小了
                n_str[i] = str(min(9,int(n_str[i+1])
                n_str[i-1] = str(int(n_str[i-1])-1)#前面的要-1
        return int("".join(n_str))
```

```python 
class Solution:
    def monotoneIncreasingDigits(self, n: int) -> int:
        n_ = str(n)
        n_str = [i for i in n_]#先变成列表，字符串是不能改的
        for i in range(len(n_str)-1,0,-1):#从个位开始
            if int(n_str[i])< int(n_str[i-1]):#如果后比前小了
                n_str[i-1] = str(int(n_str[i-1])-1)#前面的要-1
                for j in range(i,len(n_str)):#吧后面的全部变成9
                    n_str[j] = "9"
        return int("".join(n_str))
```
