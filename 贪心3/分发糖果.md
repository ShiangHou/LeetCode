## 初步思路

题目说，n个孩子，每个孩子有一个评分，然后需要给孩子糖果，每个孩子至少一个糖果，并且，相邻的孩纸中，评分高的应该得到更多的糖果，问最少的分糖果的数量是多少

首先这题，初步有这种思路，如果看完左再看右的话有些复杂，那么我们可以先看右，再看左，即：

首先先都给一个

我先右，遍历每一个孩子，看它的右边孩子的评分，如果右边孩子的评分比自己高，就多给右边孩子一个，如果右边孩子比自己低，就给自己一个

看有右孩子的话是遍历0到n-2

遍历完后再看所有人的左边，同理

看左孩子的话是遍历1到n-1

最后sum一下candy数组就行

等一下，其实不用重复，即看右边的时候，如果右边比自己打，再给右边加，右边没有自己大就不管（留着看左边的时候再加）

此外，看所有人的左的时候应该是倒着遍历把，要不然会出现错误

```python 
    def candy(self, ratings: List[int]) -> int:
        n = len(ratings)
        candy_nums = [1]*n
        #先看所有人的右
        for i in range(n-1):
            if ratings[i+1]> ratings[i]:
                candy_nums[i+1] += 1
        
        #再看所有人的左
        for i in range(n-1,0,-1):
            if ratings[i-1] > ratings[i]:
                candy_nums[i-1] += 1
        return sum(candy_nums)

```

注意，题目中说，相邻两个孩子，如果评分一样，没说必须要大，也可也小？


此外，这个写的还是有点问题，不应该单纯的+1，而是比其他的要+1

此外，左右扫的时候会有冲突，即第二次的时候需要去自己当前的值和第二次+1的最大值才行


```python     def candy(self, ratings: List[int]) -> int:
        n = len(ratings)
        candy_nums = [1]*n
        #先看所有人的右
        for i in range(n-1):
            if ratings[i+1]> ratings[i]:
                candy_nums[i+1] = candy_nums[i] +1
        
        #再看所有人的左
        for i in range(n-1,0,-1):
            if ratings[i-1] > ratings[i]:
                candy_nums[i-1] =max(candy_nums[i]+1,candy_nums[i-1])
        return sum(candy_nums)
```
