## 初步思路
看来左神讲的前序和中序去遍历，自己来试一下写出来从中序和后序

首先对于后序遍历而言，是左右中，那么改树的头节点是后序遍历的最后一个，然后拿到这个节点值去中序中找就行，由于中序是左中右，那么找到中后，中前面的就是左，然后递归遍历就行

注意，题目中说的是序列的值都不同，这避免了我从中序中找不到

这个f的递归应该这么写，首先入参两个数组，以及l1，r1，l2，r2的范围假设l2 r2是这个后序的，


首先，对于后序来说，他的head肯定是r2的位置那么

中序[l2,.......k........r2]

后序[l1,......k-1.k.........r1]

那么，左边对应着 （l2，k-1），（l1，l1+（k-l2-1)）

右边对应着（k+1，r2）（l1+k-l2,r-1）

然后不断递归就行


```python 
    def buildTree(self, inorder: List[int], postorder: List[int]) -> Optional[TreeNode]:
        hash_map = {}
        for i in range(len(inorder)):
            hash_map[inorder[i]] = i
        return self.f(inorder,0,len(inorder)-1,postorder,0,len(postorder)-1,hash_map)

    def f(self,inorder,l2,r2,postorder,l1,r1,hash_map):
        if l1>r1:
            return None
        head = TreeNode(postorder[r1])
        if l1 == r1:
            return head
        k = hash_map[postorder[r1]]
        head.left = self.f(inorder,l2,k-1,postorder,l1,l1+k-l2-1,hash_map)
        head.right = self.f(inorder,k+1,r2,postorder,l1+k-l2,r1-1,hash_map)
        return head

```

注意这里要用相对位置
