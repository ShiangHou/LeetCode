# 岛屿数量广搜

https://kamacoder.com/problempage.php?pid=1171

## 初步思路

首先需要明白什么是广搜，广搜的意思就是说，我是一层一层去搜的，并且这一层不搜完不能到下一层

在这个岛屿数量中，我们之前用的是深度搜索，目的还是要把遍历到的陆地，以及相连的陆地都打上True

那么我们用广搜其实也是一样的，也是为了把遍历到的陆地以及相邻的陆地都打上True

具体的做法是，我事先准备一个队列，然后如果遍历到了一个陆地，进入我的bfs

怎么做呢，如果是陆地，先把它扔进队列中，然后弹出

在每一个弹出的时候，遍历上下左右四个方向，把周围也是陆地的都扔进队列中，

然后依次重复，直到队列为空

队列为空意味着一一大片的陆地都被扔进过队列

弹出的时候记得标记visited，然后继续主遍历就行


```python 
n,m = map(int,input().split())


#准备好图
graph = []
for i in range(n):
    graph.append(list(map(int,input().split())))

#准备好方向
direction = [[1,0],[0,1],[-1,0],[0,-1]]

#准备好一个栈用来存放
from collections import deque
dq = deque()

#准备一个visited
visited = [[False]*m for _ in range(n)]

def bfs(graph,i,j,visited):
    #主程序已经判断过了，进来的肯定都是
    #先加入队列
    dq.append([i,j])
    #要加入的时候就标记
    visited[i][j] = True
    while dq:
        #弹出队列
        x,y = dq.popleft()
        #把当前弹出的的上下左右是1并且还是False的都放入队列
        for a,b in direction:
            next_x,next_y = x+a,y+b
            if next_x<0 or next_x>n-1 or next_y<0 or next_y>m-1:#如果出界了
                continue
            elif graph[next_x][next_y] == 1 and visited[next_x][next_y] == False:#如果下一个是陆地并且还没被放如果
                dq.append([next_x,next_y])
                #加入后立即记录
                visited[next_x][next_y] = True

    #整个过程应该持续到队列不为空






#主程序
def main():
    ans = 0
    for i in range(n):
        for j in range(m):
            if graph[i][j] == 1 and visited[i][j] == False:#如果是陆地并且还没被发现
                ans +=1
                bfs(graph,i,j,visited)#丢进去
    return ans 

if __name__ == "__main__":
    print(main())
```

纠正一点错误，一定要加入后立即标记为True，因为可能会有重叠的，如果出的时候再标记，那么重叠的可能会一直没有标记，陷入超时
