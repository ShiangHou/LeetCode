## 初步思路

这道题说，给了我们一些图，然后呢，问我们能不能从1号节点开始出发，到达任何的节点，如果可以的话，返回1，否则返回-1

这里首先和岛屿数量的建图就不一样了，之前就是坐标，现在是真正的邻接矩阵，

这里的想法其实我感觉很简单，就是不走回头路

比如说我到达了2点，那么我就是到达了，直接标记2已经到达，然后再从2开始往下走就行

那么会不会出现，我到达2之后，我另一路需要经过2，比如1 2 3抵达3，但是我想要到达4需要1 2 4呢，其实不用，因为既然可以到达2，那么子问题就是从2开始抵达所有的点就行


所以这个就是深搜，把1可以抵达的点继续dfs就行，抵达一个点就在一个存点的集合里面把它remove，最后如果说set是空的，就是已经走完了，直接返回，不是空的就返回-1


```python 
def dfs(graph,start,node_set,n):
    if start not in node_set  :#如果遍历到的点之前遍历过了
        return 

    #还有一个条件是，如果自己没有和任何点有链接（即自己是一个子叶


    #把自己删除出去
    node_set.remove(start)

    #从图里面找到那些是和自己连的
    #还有一个条件是，如果自己没有和任何点有链接（即自己是一个子叶），也需要直接返回
    #所以弄一个变量，记录一下自己的子叶
    count = 0
    for j in range(1,n+1):
        if graph[start][j] == 1:#从start到j有一条边
            dfs(graph,j,node_set,n)
            #记录一下子叶数量
            count += 1
    #如果子叶为0，也是一个终止条件，返回就行
    if count == 0:
        return 




def main():
    n,k = map(int,input().split())
    graph = [[0]*(n+1) for _ in range(n+1)]#单向图，表示从i到j有一条边,0可以不用

    node_set = set()#用来存放点集合，后续遍历一个就remove一个
    #这里点单独放，因为有的点可能没有主动连接别人
    for i in range(1,n+1):
        node_set.add(i)


    for _ in range(k):
        i,j = map(int,input().split())
        graph[i][j] = 1#从i到j有一条边

    #执行深搜
    dfs(graph,1,node_set,n)
    return -1 if node_set else 1

if __name__ == "__main__":
    print(main())
```
