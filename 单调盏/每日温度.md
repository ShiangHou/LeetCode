# 每日温度
力扣739

## 初步思路

说是给了一个数组，表示每天的温度

然后问，对于第i天，几天后温度比他高？

比如73,74,75,71,69,72,76,73

i=0的时候，发现第二天就比自己高，所以就是1天后，就是1

最后一个72，一天后的76也是比自己大，所以也是1

最后返回这个“每个i后哪一天比自己大”就行了

这里有一个初步的想法，因为要比的是后面的，所以说我们先从后往前遍历，当然第一个就是0

我们维护一个栈，栈里面是从后往前的最大值

当进来一个数的时候，和里面的比，如果比里面的大，就直接顶替进去

如果没有栈里面的大，说明栈里面的数是比自己大的，那么直接返回这个下标到自己就行

哎哎哎不对，

从后往前是73，然后76进来，压倒，弄成0，继续72，72没有里面的大，弄成76对应的位置，然后69，发现会错位，因为比69大的是72哈哈哈

所以是不是可以正着遍历就行，栈是左边的最大

先是73进去，然后进来一个74，74比它大，那么直接顶入74，73的是74的那个下标，然后继续75，顶，
当进来的是71，没有人家大，

嘶，思路有点乱


## 答案思路
重新回顾一下题目，我要找的其实就是i到n上的，比nums[i]大的，那么我们之前制定的最大和最小都不行，因为主要是要和nums[i]相比

我们来看 [75, 71, 69, 72] 这一段具体发生了什么：

75 进场： 它是目前唯一的温度，不知道明天热不热，先乖乖进候车室坐下等着。

71 进场： 它往候车室看了一眼，发现今天的 71 比候车室里的 75 还要冷。75 没等到升温，71 当然也没等到，于是 71 也在候车室坐下排队。

69 进场： 比 71 还冷，继续坐下排队。

此时候车室的队伍：[75, 71, 69]。发现规律了吗？候车室里的人，温度是严格递减的！大家都在苦苦等待变暖的那一天。

关键转折点，72 进场了！
这时候，72 站在候车室门口开始往里看（从队尾开始）：

看向队尾的 69：“我比你热！” 69 终于等到了它的升温日，拿到答案美滋滋地离开了候车室（出栈）。

继续看新的队尾 71：“我也比你热！” 71 也拿到了它的答案，离开了候车室（出栈）。

最后看 75：“哦，没你热，打扰了。”

处理完别人后，72 自己因为也不知道未来的天气，也走进了候车室坐下。此时候车室变成了 [75, 72]，依旧保持递减！h 


那么问题是，怎么存下标呢，是连着数字和下标都存嘛，

我们可以在栈里面只存下标就行了，然后直接那temperatures取数就行了

```python 
class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        stack = [0]#一个单调递减的栈
        n = len(temperatures)
        ans = [0]*n
        for i in range(1,n):
            while stack and temperatures[i]> temperatures[stack[-1]]:#如果有一个更大的了,那么我们把栈顶的弹出，并且记录，
                ans[stack[-1]] = i-stack[-1]
                stack.pop()
            #当都没有的时候，直接把自己加入
            stack.append(i)
        #最后把stack里面剩的都弄成0
        while stack:
            ans[stack[-1]] = 0
            stack.pop()
        return ans


```
