## 初步思路

首先要知道什么是二叉搜索树

所谓的二叉搜索树，是指，对于一个节点来说，他的左边的值是小于他自己的，右边的值是大于他自己的，同时左右子树也满足这个，我们把这个成为二叉搜索树

Gemini说，二叉搜索树的插入、删除、平均查找的时间复杂度都是O(logn),因为每次可以斩杀一半

回到题目中，题目让我们找到那个节点等于val的，并返回以它为root的子树

思路very的简单啊，首先看根节点是不是，如果不是的化，看看大了还是小了，大了就往右边走，小了就往左边走，直到最后返回不就行了

```python 
class Solution:
    def searchBST(self, root: Optional[TreeNode], val: int) -> Optional[TreeNode]:
        if not root:
            return 
        if root.val == val:
            return root
        elif root.val > val:#说明这个节点大了，答案在左边
            return self.searchBST(root.left,val)
        else:
            return self.searchBST(root.right,val)


```
