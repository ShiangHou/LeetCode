# 非递减子序列

力扣491

## 初步思路

首先这道题说是找到所有的非递减子序列，看了一眼样例

对于数组[4,6,7,7]

那么对于 4 6 是子序列，同理 4 6 7 7  

同时，4，7 也是子序列

分析一下可以发现，这道题跟我们之前做的很想，即要找的其实都是子集

比如从4开始往后实验，46开始往后实验，符合条件就一直加，不符合条件就跳，再从4 7开始

那么这里依旧是写一个递归函数

f(start,path)

终止条件是path走到头了，我们把path加一下就行

然后写一个for循环

啊啊啊好难写啊，思路首先是先找到i下面的那个最大的，发现无法处理i

所以说还是要把头写在for里面，但怎么判断前后的大小关系呢




## 最终思路

```python 
def findSubsequences(self, nums: List[int]) -> List[List[int]]:
    self.ans = []
    self.f(0,nums,[])
    return self.ans

def f(self,start,nums,path):
    if len(path) >= 2:
        self.ans.append(path.copy())
    used = set()
    #然后从j开始递归
    for i in range(start,len(nums)):
        if path and nums[i] < path[-1]:#如果当前这个比path的最后一个还小，直接跳过
            continue
        if nums[i] in used:
            continue
        used.add(nums[i])
        path.append(nums[i])
        self.f(i+1,nums,path)
        path.pop()  
```
这里有几个点需要清晰一下，首先是判断，我们可以直接用path的最后一个来去判断就行，这样的话，可以很好的避免住处理头的情况

其次，我们的for一定要是从头去写，不从头去写，后面就pop不出第一个头，导致每一个都是从头开始的


最后，为了避免 4 7 6 7 ，在四的时候，我们会选择两边 4 7 ，需要去重，如果这一层有数字被用过了，直接跳过就行
