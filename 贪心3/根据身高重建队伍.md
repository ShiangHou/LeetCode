## 初步思路

题目说一个人有两种属性

一个是所谓的  前面有多少人比自己高


但是目前的数组并不是按照 前面有多少人比自己高 这样去排列的

现在题目要求我们把队伍整理起来，使队伍符合这个要求

## 答案思路

答案的说的其实也没太听懂

首先我们需要把这些人按照身高从大往小去排列

然后，开始按照k的值去往新的list里面去插入就行

这是因为，我们先排序了，有一个小细节，矮的人插入到前面后不会影响到高的人

此外，由于我们是按照身高排序的，所以最高的那个肯定是 每人比自己前面更高，所以是0

假设有四个人，已经按身高降序排列（身高相同时，$k$ 值小的排前面）：

[7,0], [7,1], [6,1], [5,0]。

我们准备一个空列表作为新队伍，依次执行插入操作：

处理 [7,0]：队伍为空。他需要前面有 0 个人比他高，直接插入索引 0。当前队伍：[[7,0]]

处理 [7,1]：队伍中已有的人都大于等于他。他需要前面有 1 个人，直接插入索引 1。当前队伍：[[7,0], [7,1]]

处理 [6,1]：队伍中已有的人（7和7）都大于他。他需要前面有 1 个人，直接插入索引 1。原索引 1 及其后面的元素会自动后移。当前队伍：[[7,0], [6,1], [7,1]]

处理 [5,0]：队伍中已有的人都大于他。他需要前面有 0 个人，直接插入索引 0。当前队伍：[[5,0], [7,0], [6,1], [7,1]]

所以我们只需要先按照身高排序，然后遍历一下，按照person直接插入到列表中就行（要求挤入后，后面的元素依次后移一位

有两个python知识，一个是排序，一个是插入还要后后移

先说排序

python中的sort函数可以key = 

people.sort(key=lambda x: (-x[0], x[1]))

key是一个关键字参数，它接收一个可调用对象（通常是函数）。在排序执行期间，底层排序算法（Timsort）会将列表中的每一个元素作为参数传递给该函数，并根据函数返回的值来决定元素之间的绝对顺序

lambda x 是一个匿名函数

简单来说，比如我有一个数列，[7,4]

我想写一个函数，返回这个数列的0位置的元素

那么正常来写的话就是 def ，然后入参 return 0位置

但是lambda可以这么写
lambda x : x[0]

同理这里的意思就是

people.sort(key=lambda x: (-x[0], x[1]))

我要key ，这个key是什么呢，就是x数组的0位置和1位置，所以就写成这样

然后第二个问题就是，插入数组

这里可以直接insert就行


```python 
class Solution:
    def reconstructQueue(self, people: List[List[int]]) -> List[List[int]]:
        people.sort(key = lambda x: (-x[0],x[1]))
        queue = []
        for person in people:
            queue.insert(person[1],person)
        return queue

```
