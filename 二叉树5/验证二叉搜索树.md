## 初步思路

在刚才我们学习了二叉搜索树，值得注意的是，对于一个node，如果它的右节点的左节点，也要严格大于该node，即有一个区间的概念

换句话说，对于这题来讲，当我们在一个node的时候，要用这个node的val来当作：

所有左子树的最大值

所有右子树的最小值

同时在往下递归的时候也要保证

梳理一下，对于一个node

判断左节点的val只需要 小于 node.val就行，往下递归的时候传入一个区间变量[min,max](对于第一个的左，就是[0,node.val]),其中0代表着下面这波的node的左右子节点的范围

到左子节点时（假设是node），看左子节点 的 左节点，他应该是 0-node.val,右节点是 node.val，max

同理一直往下递归传递就行

在递归函数中，node本身是不用判断的，因为node本身是在上一层被判断的


```python 
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        self.flag = True
        self.check(root,float('-inf'),float('inf'))
        return self.flag
        
    def check(self,node,_min,_max):
        if not node:
            return 
        if node.left:
            if _min>=node.left.val or node.left.val >=node.val:#小过头了或者不比父节点大
                self.flag = False
            else:#如果正常，就继续往下走,这是左子树，所以更新后的max就是node.val
                self.check(node.left,_min,node.val)
        if node.right:
            if node.right.val >= _max or node.right.val <= node.val:
                self.flag = False
            else:
                self.check(node.right,node.val,_max)
```

注意一点，这个node只要求严格大于或者小于，所以要把等于弄到false里面

## 优化

其实这样是有问题的，即我在一个node去检查它的左右孩子

最好的方法是，我去检查这个node的本身行不行，然后传参也是传参这个node本身

此外，最好要会写这种True迭代然后接住的，不要弄这种全局变量

```python 
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        return self.check(root,float('-inf'),float('inf'))
        
    def check(self,node,min_val,max_val):
        if not node:
            return True #检查到底了，说明这一路下来都是对的
        if not (min_val< node.val< max_val):
            return False
        return self.check(node.left,min_val,node.val) and self.check(node.right,node.val,max_val)#即左右子节点是不是在范围内，顺便更新了范围

```

注意一下 and是有一个False都是False，or是有一个True都是True，然后not node注意是返回True
