## 初步思路
依旧是有一点记忆

首先是需要有一个统计队列的长度n

入队列的时候，压入队列，同时弹出队列并压入队尾，这个过程持续n次，然后n再n+1

弹出的时候正常弹出

peek也是正常

在python中，collections导入deque就行

```python 
class MyStack:
    def __init__(self):
        from collections import deque
        self.deque = deque()
        self.size = 0
    def push(self, x: int) -> None:
        self.deque.append(x)
        for i in range(self.size):
            temp = self.deque.popleft()
            self.deque.append(temp)
        self.size += 1
    def pop(self) -> int:
        self.size -= 1
        return self.deque.popleft()
    def top(self) -> int:
        if self.deque:
            return self.deque[0]
        else:
            False
    def empty(self) -> bool:
        return self.size == 0
```
