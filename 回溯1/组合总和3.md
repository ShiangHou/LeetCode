## 思路

和组合那道题差不多，依旧是在脑海里想象一个二叉树，要还是不要，然后遍历所有

直到抵达条件（总和==target）或者超出限制，就返回就可以

```python 
    def combinationSum3(self, k: int, n: int) -> List[List[int]]:
        self.ans = []
        self.max_len = k
        self.f(1,k,n,[])
        return self.ans
        
    def f(self,i,k,target,path):
        if target == 0 and k == 0:#如果搜集到了目标，就返回
            self.ans.append(path.copy())
            return 
        if k == 0 and target != 0:#到达k个上线了，但是还没有搜集到
            return 
        if i > 9:#只能用0-9，如果超过也返回
            return 
        #用了i
        path.append(i)
        self.f(i+1,k-1,target-i,path)
        #删除i
        path.pop()
        #没用i
        self.f(i+1,k,target,path)
        return path

```
注意题目要求是k也要正好用完，且target也要正弄完
