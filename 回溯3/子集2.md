## 初步思路

虽然题目中说是子集，但本质上求的还是组合啊，和我们在组合总和2里面是一样的，只不过不要求是总和了，就是单纯的组合

那么首先依旧是先把数组进行排序，为了方便去重，

递归函数f(i,s,path) 

由于这里不用求和了，所以递归的结果就是i到len了
排

比如1 1 1 1 2 2 2

排完序后，思路就是，“1”我拿几个，比如拿0个，拿1个，拿2个等等，然后直接往下递归到2位置上去就行

```python 
class Solution:
    def subsetsWithDup(self, nums: List[int]) -> List[List[int]]:
        self.ans = []
        nums.sort()
        self.f(0,nums,[])
        return self.ans

    def f(self,i,nums, path):
        if i  == len(nums):#终止条件，看i走到头
            self.ans.append(path.copy())
            return 
        #找到下一个新的位置
        j = i + 1
        while j <(len(nums)-1) and nums[j] == nums[j+1]:
            j += 1
        #当前这个位置不取
        self.f(j,nums,path)

        #当前这个i位置分别取取0-（j-i）个，然后每一个都循环递归
        for k in range(i,j):
            path.append(nums[k])
            self.f(j,nums,path)
        path.pop()
        return 
        

```

有几个问题，首先nums[j]应该直接和nums[i]去比，和当前这一层不一样就行，并且j要从i开始

其次，这一个
j <(len(nums)-1) 也有点问题，这里如果i是等于最后一个的，那么这个条件就会永远判定不对，那么j就无法往下走，从而死循环

此外最后的pop也是，刚才j-i进去了几个，最后就pop出几个



## 正确答案

```python 
class Solution:
    def subsetsWithDup(self, nums: List[int]) -> List[List[int]]:
        self.ans = []
        nums.sort()
        self.f(0,nums,[])
        return self.ans

    def f(self,i,nums, path):
        if i  == len(nums):#终止条件，看i走到头
            self.ans.append(path.copy())
            return 
        #找到下一个新的位置
        j = i
        while j <len(nums) and nums[j] == nums[i]:#是j<len,如果是-1，那么会无限循环
            j += 1
        #当前这个位置不取，即i位置取0个
        self.f(j,nums,path)

        #当前这个i位置分别取取1-（j-i）个，然后每一个都循环递归
        for k in range(i,j):
            path.append(nums[k])
            self.f(j,nums,path)
        
        #这里全部弹出，刚才进去了有j-i个
        k = j-i
        while k >0:
            path.pop()
            k-=1


```
当然，这个思路是左程云的，我们可以按照组合总和来去写，就是往下走的时候，如果遇到相同的就continue就行，也是very的清爽，最后只需要pop一下就行，左程云用的是size来去控制path，道理是一样的
