# 爬楼梯

力扣70

## 初步思路
上楼梯一次可以上1个或者2个，问有多少种方式上n的楼梯

首先当我到第i个楼梯的时候，可能是从i-1上来的，也有可能是从i-2上来的，

如果i是1，那么就1种，如果i是2，那么就是两种


```python 
class Solution:
    def climbStairs(self, n: int) -> int:
        if n == 1:
            return 1
        if n == 2:
            return 2
        return self.climbStairs(n-1) +self.climbStairs(n-2)
```

不行，这样递归会超时

为什么会超时呢，因为会重复计算

我们加一个记忆搜索吧，我们可以弄一个数组，先把数都存在里面，然后递归再调用的时候直接拿就行


```python 
class Solution:
    def climbStairs(self, n: int) -> int:
        return self.find(n,[-1]*(n+1))

    def find(self,n,dp):#这个dp初始化为全是-1的，长度是n
        if dp[n] != -1:
            return dp[n]
        if n == 1:
            dp[1] =  1
            return dp[n]
        if n == 2:
            dp[2]  = 2
            return dp[n]
        dp[n] = self.find(n - 1, dp) + self.find(n - 2, dp)
        return dp[n]
```

分析一下，其实我们可以发现就是一个填表的过程，dp表位置的1和2位置填1和2，然后后面就一直填这个dp表就行

```python

class Solution:
    def climbStairs(self, n: int) -> int:
        if n == 1:
            return 1
        if n == 2:
            return 2
        dp = [-1]*(n+1)
        dp[1] = 1
        dp[2] = 2
        for i in range(3,n+1):
            dp[i] = dp[i-1]+dp[i-2]
        return dp[n]
```
