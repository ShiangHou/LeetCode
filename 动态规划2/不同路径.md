# 不同路径

力扣62


## 初步思路

题目说一个m*n的小网格，然后机器人从0 0出发，只能往右或者往下走，问走到m-1 n-1的位置上有多少种走法

这题就是自然的动态规划，因为只能往下和右走，所以对于一个节点来说，他的来源只能是上和右

所以dp[i]的含义是到达i的次数，那么就等于上面来的加上左边来的

即 dp[i][j] = dp[i-1][j] + dp[i][j-1]

开始填空，对于第一个行，只能填1，因为只能从右边走，同理，对于第一列，也只能填1，因为只能从上面来

```python 
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[0]*n for _ in range(m)]

        for i in range(n):
            dp[0][i] = 1
        for j in range(m):
            dp[j][0] = 1
        
        for i in range(1,m):
            for j in range(1,n):
                dp[i][j] = dp[i-1][j] + dp[i][j-1]
        return dp[m-1][n-1]
        

```
