# 最佳买卖股票 含手续费 

力扣714

## 初步思路

这个感觉就是和之前是一样的，可以发现，一买一卖收一次，那就再卖的时候加上就好了


```python 
class Solution:
    def maxProfit(self, prices: List[int], fee: int) -> int:
        n = len(prices)
        dp = [[float('-inf')]*2 for _ in range(n)]
        dp[0][0] = 0
        dp[0][1] = -prices[0]
        for i in range(1,n):
            dp[i][0] = max(dp[i-1][0],dp[i-1][1]+prices[i]-fee)
            dp[i][1] = max(dp[i-1][1],dp[i-1][0]-prices[i])
        return dp[n-1][0]
```
