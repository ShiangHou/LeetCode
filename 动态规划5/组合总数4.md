## 初步思路

题目说给我们一个数组（数组由不同的整数组合而成），问找出怎么从数组中的组合=target的有多少种

看了一下例子，可以重复取，那就是一个标准的，可以重复取的完全背包

```python 
class Solution:
    def combinationSum4(self, nums: List[int], target: int) -> int:
        dp = [0]*(target+1)
        n = len(nums)
        dp[0] = 1#含义是取多少是0,这里是1,即不取
        for num in nums:
            for i in range(target,num-1,-1):
                dp[i] = dp[i-num]
        return dp[target]
```

这样写是不对的，为什么？



## 压缩技巧

回看我们之前的（用纸和笔画一画），在01背包中，我们i和j只依赖于i-1那一行，因此当我们从后往前便利的时候，前面其实就是我们的i-1行的位置，

然后完全背包中，我们不仅依赖于i-1的行，还依赖于i行，换句话说，当我们更新i j的时候，除了依赖自身以外，还依赖自己前面已经更新好的值，当我们从后往前的时候，自然就不行

那怎么办呢，简单，刚才不是说后面的要依赖前面的嘛，那我们就先更新前面的呗，


```python 
class Solution:
    def combinationSum4(self, nums: List[int], target: int) -> int:
        dp = [0]*(target+1)
        n = len(nums)
        dp[0] = 1#含义是取多少是0,这里是1,即不取
        for j in range(1,target+1):
            for num in nums:
                if j-num >=0:
                    dp[j] += dp[j-num]#自己现在就是i-1的位置，然后如果前面的有就加上（也就是不变），没有就是自己

        return dp[target]


```
