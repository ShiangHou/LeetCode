# 组合总和2

力扣40

## 思路
和组合总和1是一样的，不过要求每个数只能用一次，

```python 
class Solution:
    def combinationSum2(self, candidates: List[int], target: int) -> List[List[int]]:
        self.ans = set()
        self.f(candidates,target,0,[])
        return [list(x) for x in self.ans]
    def f(self,nums,target,i,path):#i是在nums下滑动的下标，path是单个搜集的路径
        if target < 0:
            return 
        if target == 0:
            self.ans.add(tuple(path))
            return path
        if i >= len(nums):
            return 
        #用i,但用了就不能继续用了
        path.append(nums[i])
        self.f(nums,target-nums[i],i+1,path)

        #弹出
        path.pop()

        #不用i
        self.f(nums,target,i+1,path)

        return path
```
这么写其实有两个问题，首先是去重，比如我先遍历了(1,7) 后面又找到了(7,1)，这样对于一个tuple来说还是不同的，

解决方案是，我们提前吧nums处理一下，避免这种路径。 

处理方式就是sort排序一下

第二个问题,超时，当全都是1的时候，会一直走，我么需要剪一下

方法很简单，当i+1和i相同的时候，我们直接跳过就行


```python 
    def combinationSum2(self, candidates: List[int], target: int) -> List[List[int]]:
        self.ans = set()
        candidates.sort()
        self.f(candidates,target,0,[])
        return [list(x) for x in self.ans]
    def f(self,nums,target,i,path):#i是在nums下滑动的下标，path是单个搜集的路径
        if target < 0:
            return 
        if target == 0:
            self.ans.add(tuple(path))
            return path
        if i >= len(nums):
            return 
        #用i,但用了就不能继续用了
        path.append(nums[i])
        self.f(nums,target-nums[i],i+1,path)

        #弹出
        path.pop()

        #不用i
        #当下一个i也是相同的时候，往后退
        while i<len(nums)-1 and nums[i] == nums[i+1]:
            i+=1
        self.f(nums,target,i+1,path)

        return path
```
