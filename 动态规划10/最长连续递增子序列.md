## 初步思路
这个看起来比较简单，要求是递增的，并且是连续的，那么我们可以直接写，首先都是1，然后填表

因为只要前面的i-1比自己大，那么说明在自己这断了，重制就行哈哈哈

就是遍历一遍，如果i-1比自己小，就续上，+1，如果比自己大，那么就设置为1就行，最后取整个dp最大的那个

```python 
class Solution:
    def findLengthOfLCIS(self, nums: List[int]) -> int:
        n = len(nums)
        dp = [1]*n
        for i in range(1,n):
            if nums[i]> nums[i-1]:
                dp[i] = dp[i-1]+1
            else:
                dp[i] = 1
        return max(dp)
```

丝滑通过，遍历一遍，时间复杂度是n
