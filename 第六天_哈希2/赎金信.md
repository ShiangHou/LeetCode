## 初步思路
这个题目，感觉和那个异位词很像啊，都是看是不是完全组成的
那依旧是用我们的ASCII码来去做，先减去97，然后弄一个26位的，原数组+1，后面的-1，最后看是不是全为0就行，

嗷嗷注意一下，并不是看全不全是0，因为分析题意，只需要覆盖就行，那么最后就看有没有1就行，只要没有1，就说明源字符串已经被完全覆盖了

嗷嗷不对，是看有没有大于0的，因为如果重复，并且覆盖了，但覆盖不全，就会出现只有几个的

```python 
def canConstruct(self, ransomNote: str, magazine: str) -> bool:
    n_nums =[0]*26
    for i in ransomNote:
        n_nums[ord(i)-97] += 1
    for j in magazine:
        n_nums[ord(j)-97] -=1
    for k in n_nums:
        if k > 0:
            return False
    return True

```
