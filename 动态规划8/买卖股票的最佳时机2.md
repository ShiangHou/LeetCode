## 初步思路

这题之前贪心的时候做过，遍历一遍，然后i+1如果大于i，把差值给全部统计进去就行


## dp思路

回归dp，沿用我们刚才的思路可以嘛，试一试，

初始状态还是不变的，思考动态规划

dp[i][0] ,当前状态没持有，那么它继承i-1时刻，可能是i-1也没持有，也有可能是i-1持有但是现在卖掉了，

max(dp[i-1][0],dp[i-1][1]+price[i])

dp[i][1] 当前状态是持有的，有可能i-1也是持有的，或者i-1没持有，现在买了，

因为这是可以无限买卖，那么就是

max(dp[i-1][1],dp[i-1][0]-price[i])

这里就是我们在买卖股票1的时候做错的那个方式

```python 
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        n = len(prices)
        dp = [[0]*2 for _ in range(n)]
        dp[0][1] = -prices[0]
        for i in range(1,n):
            dp[i][0] = max( dp[i-1][0],dp[i-1][1]+prices[i])
            dp[i][1] = max( dp[i-1][1],dp[i-1][0]-prices[i])
        return dp[n-1][0]

```
