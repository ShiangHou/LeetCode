## 初步思路
题目问，说给我们一个整数，用这个1-n的整数组成二叉搜索树有多少种？

有点懵，二叉搜索树就是说，一个节点的左节点比自己小，右节点比自己大

回想我们之前写过的构建二叉搜索树，是直接取二分之一，然后再加的

先想一个暴力的把

首先我先想，我用这个1个，那么就是一种

我用1和2，那么就是两种，其中以1为根的有1个，然后以2为根的有1个

用1和2和3呢，首先先看1，就是一种

然后我们看用1和2的，就像刚才所说的，有2钟

然后继续看3，当分析到自己的时候，

1：之前的那些由于都是比自己小的，所以自己可以当头，之前所有的都是自己的左节点，（然后之前的内部有2种，所以这里是2）

2：如果自己不当头，让之前的当头，那么自己是在最尾巴，那么之前其实一共也是两种，那么这里也是2种

3：让之前的当头，但是自己不是尾巴（但自己一定是最后一颗右子树）

## 暴力递归思路

之前的想反了，其实应该从**谁来当root**去考虑

对于小于n的，我们可以发现，如果取i作为root，那么 0-i的只能构成左子树，i+1到n只能构成右子树，

此外，要注意，1 2 3和 4 5 6 这两个序列可以构成的子树的数量是一样的！换句话说，树的拓扑结构只与需要构成的数量有关，而与具体的值是无关的，

因此，如果一个序列 1 2 3 4 5 6，我选择1当root，那么后面的2 3 4 5 6继续递归就行

我选择3当root，那么左边就是 1 2递归，右边就是4 5 6递归，两边的种类相乘就是这一个的情况

最后，我需要把这些全部都加起来

所以Fn的含义就是，n个节点能够构建的树的种类，然后不断递归就行


```python 
class Solution:
    def numTrees(self, n: int) -> int:
        if n == 0 or n == 1:#0个节点和1个节点都是0种
            return 1
        _sum = 0
        for i in range(1,n+1):
            _sum += self.numTrees(i-1)*self.numTrees(n-i)
        return _sum

```

```python 
#记忆化搜索
class Solution:
    def numTrees(self, n: int) -> int:
        dp = [-1]*(n+1)
        return self.f(n,dp)
    
    def f(self,n,dp):
        if dp[n] != -1:
            return dp[n]
        if n == 0 or n == 1:
            dp[n] = 1
            return dp[n]
        temp = 0
        for i in range(1,n+1):
            temp += self.f(i-1,dp)*self.f(n-i,dp)
        dp[n] = temp
        return dp[n]

```

## 动态规划

看到了，dp的意思就是，


```python 
class Solution:
    def numTrees(self, n: int) -> int:
        dp = [0]*(n+1)
        dp[0],dp[1] = 1,1
        for i in range(2,n+1):
            for j in range(1,i+1):
                dp[i] += dp[j-1]*dp[i-j]
        return dp[n]
```
