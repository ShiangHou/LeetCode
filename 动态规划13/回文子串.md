## 初步思路

这题说的是找到所有的回文的子串

首先思路暴力的是，找到所有的子串，然后判断是不是回文的

这里要求子串是连续的，那我们其实可以弄一个递推的

dp[i][j]表示以下标开始，以下标j结尾是不是，

然后我们遍历到i的时候，看看nums[i:j]是不是回文，如果是的话，该位置直接变成1，不是的话变成0，最后我们取一个ans最后都ans+1就行

```python 

class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        # dp = [[1]*n for _ in range(n)] 这个甚至没用到
        ans = 0
        for i in range(n):
            for j in range(i,n):
                if self.judge(s,i,j):
                    ans+=1
        return ans   

    def judge(self,s,i,j) -> bool:
        sub_s = s[i:j+1]
        return  sub_s == sub_s[::-1]
```
吗的居然过了，而且好像没用dp表哈哈哈哈，时间复杂度是n2，但是时间还是比较爆的

其实不是，厚米说
但我们严谨地盘一下你现在的复杂度：两层 for 循环是 $O(N^2)$，里面每次切片 s[i:j+1] 加上翻转比较又需要遍历一次子串，最坏情况是 $O(N)$。所以你现在的真实时间复杂度其实是 $O(N^3)$

## 优化思路

整体思路还是对的，即dp[i][j]表示以下标开始，以下标j结尾是不是回文

但是这里我们需要换一个角度，我们的j要从最后开始！，即往里面夹

如果i位置和j位置是一样的，那么我们需要依赖里面的i+1位置和j-1的位置，

这就要求我们开始填的时候从里面往外面填，可以发现，j是依赖前面的，然后i依赖后面，立即推，j是从0开始遍历，然后i是从n-1开始往后遍历，

哎哎哎不对，ij是从i开始到j结尾，i依赖的是i+1的位置，i从n-1开始，然后j是从i到n-1开始遍历，这样就行了，不会错过了，因为如果j从0开始，那么就会出现ji交叉的情况，这样的话就是需要

那dp表里面填什么，还是数字嘛，可以填True和False，这样用一个and就行,最后统计True的数量就行
```python 
class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        dp = [[False]*n for _ in range(n)]
        #意思是ij框起来的是不是回文，i依赖i+1，j依赖j-1
        ans = 0
        for i in range(n-1,-1,-1):
            for j in range(i,n):
                if i == j:
                    dp[i][j] = True
                    ans+=1
                elif s[i] == s[j]:
                    dp[i][j] = dp[i+1][j-1]
                    if dp[i][j] == True:
                        ans+=1
                else:
                    dp[i][j] = False
        return ans 
```

aaa为什么不对？

当 i = 0, j = 1 时，遇到了相邻的两个相同的字母。

程序进入 elif，去查表 dp[i+1][j-1]，也就是查 dp[1][0]

最好还是合并在一起写，如果相等的话，

```python 
class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        dp = [[False]*n for _ in range(n)]
        #意思是ij框起来的是不是回文，i依赖i+1，j依赖j-1
        ans = 0
        for i in range(n-1,-1,-1):
            for j in range(i,n):
                if s[i] == s[j]:#如果相同的话，需要看一下是不是
                    if j-i <=1 or dp[i+1][j-1] ==True :
                        dp[i][j] = True
                        ans+=1
        return ans 
```

如果相等的话，判断一下相差，如果是差了0（即同一个）或者相差1（即相邻），并且还相等，
