## 初步思路
一个字符串s，然后一个字符串的列表，问能不能用这个列表里面的字符串吧这个s给拼出来，列表里面的可以无限用

这里是一个多重背包问题，只不过需要考虑的是分词的问题，就是说我的字符串是一大串，怎么样去构建这个开始填包的位置和后序吃下一个后位置的跳转

后序吃下一个后位置的跳转这个就直接然后i跳到len就行，

换句话说，我的背包量每次跳到i-len(word)就行，其实比较清晰

注意，如果能往后跳的话，需要保证当前的word能够卡住s的对应的字段

```python 
class Solution:
    def wordBreak(self, s: str, wordDict: List[str]) -> bool:
        n = len(s)
        dp = [False] * (n+1)
        dp[0] = True#0个就是不用构建，就是True
        for i in range(n+1):
            for word in wordDict:
                if i >= len(word):
                    dp[i] = dp[i] or (dp[i-len(word)] and word == s[i-len(word):i])#注意，如果是dp[i-len(word)]，那么需要保证word要和s对应的位置是一样的
        return dp[n]


```
