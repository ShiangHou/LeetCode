# 沉没孤岛

https://kamacoder.com/problempage.php?pid=1174

## 初步思路

所谓的孤岛和刚才说的一样，就是上下左右四个都是水，并且不和边缘连结在一起的

我们把孤岛弄成0就行

我们沿用上题的思路，如果发现是一个孤岛，那么就用i再去搜一遍（比如说用dfs）把对应的弄成0就行

但其实有个更方便的，就是和孤岛总面积一样，从四周遍历，把四周遍历到的是1的，然后用深搜给他全部弄成2，这样剩下的图中是1的就是孤岛了，然后再遍历一遍就行



```python 
n,m = map(int,input().split())

#建岛
graph = []
for i in range(n):
    graph.append(list(map(int,input().split())))

#方向
direction =[[1,0],[-1,0],[0,1],[0,-1]]

#visited
#visited = [[False]*m for _ in range(n)]


#这个是用来深搜的，并且把深搜的结果都弄成2

#这里可以不用visited，相当于用原来的graph，把graph变成2
def dfs(graph,i,j):
    if graph[i][j] == 0 or graph[i][j] == 2:
        return 
    else:
    #先把当前变成2
        graph[i][j] = 2

        #遍历方向
        for x,y in direction:
            next_i,next_j = i+x,j+y
            if next_i<0 or next_i>n-1 or next_j <0 or next_j >m-1:
                #越界，
                continue
            if graph[next_i][next_j] == 1:#不用判断是不是visited了，因为visited会是2
                dfs(graph,next_i,next_j)#变2的操作在递归里面就会有


def main():
    #先把边缘的都dfs一遍

    #先弄，是第0行和最后一行
    for j in range(m):
        if graph[0][j] == 1:
            dfs(graph,0,j)
        if graph[n-1][j] == 1:
            dfs(graph,n-1,j)

    #在弄两边
    for i in range(n):
        if graph[i][0] == 1:
            dfs(graph,i,0)
        if graph[i][m-1] == 1:
            dfs(graph,i,m-1)
    
    #现在非孤岛已经是2了，然后遍历一遍，把孤岛弄成0，
    for i in range(n):
        for j in range(m):
            if graph[i][j] == 1:
                graph[i][j] = 0

    #最后把2还原回去
    for i in range(n):
        for j in range(m):
            if graph[i][j] == 2:
                graph[i][j] = 1
    
    #打印出去
    for raw in graph:
        print(" ".join(map(str,raw)))

if __name__ == "__main__":
    main()
```
