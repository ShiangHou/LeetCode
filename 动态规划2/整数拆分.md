## 初步思路

给一个整数，n，我可以把这个整数任意拆分（要大于等于2个），把拆好的这些数乘起来使其最大，求这个最大的乘积是多少

这里有个k，即把这个数拆分成k个数的和，这个不是很好确定

如果是10，拆完就是5和5，然后再拆成 2*2*3*3就是36，再拆，拆成2*2*2*2*2则是32，没有那个36的大，但是拆成3*3*3*1就只有27

算了，厚米提示我，让我暴力到底

首先，我需要拆分一个i出来，这个i是1到n，然后剩下的就是n-i

对于n-i，我们可以选择直接用，或者说再拆分

那么此处就是max i*(n-i) 和i*递归（n-i），即所谓的拆和不拆

梳理一下思路，

首先是，我需要遍历所有的1到n，定好一个i，然后剩下的就是n-i

然后开始调用递归，返回max(i*(n-i),i*bfs(n-i))

此外，我们需要设置一些边界条件，如果n是1的话，直接返回1就行，如果n是2，返回2 n是3 返回3 n是4返回4，就像我们刚才讨论的一样


```python 
class Solution:
    def integerBreak(self, n: int) -> int:
        if n ==2:
            return 1
        if n== 3:
            return 2
        #上面两个是正常的答案，即n就是2或者3
        
        return self.f(n)


    def f(self,n):
        if n == 1 or n == 2 or n ==3 or n == 4:#这是当n需要拆分的子问题的
            return n
        cur_max = 0

        for i in range(1,n):
            #遍历每一种组合，然后把所有的结果都存在_max里面
            cur_max = max(i*(n-i), i*self.f(n-i),cur_max)#需要不断刷这个cur_max
        return cur_max
```

n为49的时候就会超时，这个时间复杂度是2的n次方

因为有大量的重复计算，我们用一个dp表格就可以来去解决

当我们计算过一次的时候就存在dp表里，这个dp表应该是n的长度


```python 
class Solution:
    def integerBreak(self, n: int) -> int:
        if n ==2:
            return 1
        if n== 3:
            return 2
        dp = [-1]*(n+1)
        return self.f(n,dp)


    def f(self,n,dp):
        if dp[n] != -1:
            return dp[n]

        if n == 1 or n == 2 or n ==3 or n == 4:
            dp[n] = n
            return dp[n]
        cur_max = 0

        for i in range(1,n):
            #遍历每一种组合，然后把所有的结果都存在_max里面
            cur_max = max(i*(n-i), i*self.f(n-i,dp),cur_max)
        dp[n] = cur_max
        return dp[n]

```

这样就可以过了

## 动态规划

我们来看一下这个递归过程，或者说这个记忆化的表格，

首先看这个dp的含义是什么，我们看dp[n] = cur_max可以发现，dp里面存的，不就是当前n的拆分的和的乘最大值嘛，

再看遍历过程，从i出发，一直往后，

嘶，其实无论如何都要去做一下和自己后面的成

也就是说，dp[n]其实等于 max(1*dp[n-1],2*dp[n-2] 等等等)

那么就可以用N2的复杂度来去写了


```python 
class Solution:
    def integerBreak(self, n: int) -> int:
        if n ==2:
            return 1
        if n== 3:
            return 2
        dp = [-1]*(n+1)
        dp[1],dp[2],dp[3] = 1,2,3
        for i in range(4,n+1):
            cur_max = 0
            for j in range(1,i):
                cur_max = max(j*dp[i-j],cur_max)
            dp[i] = cur_max
        return dp[n]
```

这样写要从我们刚才讨论的性质说起，即任何一个大于4的最后都是其本身

当然，我们其实可以用刚才的递归函数，即


```python 
class Solution:
    def integerBreak(self, n: int) -> int:
        if n ==2:
            return 1
        if n== 3:
            return 2
        dp = [-1]*(n+1)
        dp[1],dp[2],= 1,2
        for i in range(3,n+1):
            for j in range(1,i):
                dp[i] = max(j*dp[i-j],dp[i],j*(i-j))
        return dp[n]

```

这样写可以保证按照我们的规则，3也是会输出2，而且后面的都不会变
