# 柱状图中最大的矩形

力扣84

## 初步思路

这道题说，也是给我们一个数组，这些数组的数组代表了他们的高度 ，相当于一个圆柱，然后圆柱和圆柱之间不是有相邻的嘛，相邻的可以组成矩形，问整个数组能组成的最大的矩形面积是多少

题目给的例子是2,1,5,6,2,3

这个能不能用贪心做，想一下

首先我们遍历的时候，用i

当往下走的时候，算一下加上去后面积有没有变大，或者有没有加自身

比如2，那么面积就是2

下一个是1，那么i和1组成的面积就是j-i然后再加

嘶，这样的话，我需要双层遍历了，因为最坏的情况是n2

## 思路

注意到，我们和接雨水想的一样，就是


当对于i，如果必须以i作为我的高的话，那么看一下左右两侧最宽能到多少

2,1,5,6,2,3

比如2，如果以2最高，那么左边没有，右边是一个1，被卡住了,那么以2为高只能是自己

找那个被卡住的，实际上就是再找右边第一个比自己小的

同理，我们也需要找，左边第一个比自己小的

怎么找左右两端比自己小的，我们在每日温度里面有说过，这个东西是在做栈的时候动态求的

先看右边第一个比自己小的，这个因为是右边，所以是顺序遍历

2,1,5,6,2,3

2先进栈，

然后是1，1要和2去比，比2小，1直接强势杀入，2滚蛋，那么2右边第一个比自己小的就是1

然后是5，5比1大，继续，然后是6，6比5大，继续，压入

然后是2，现在栈里面是1 5 6 2进来后，6走，记录，然后5走，记录，然后以此类推

这样就得到了右边第一个比自己小的，这里记录下标就行

同理，反着遍历可以找到左边比自己小的，


嗷嗷，其实不需要反着遍历，因为左边比自己小的本来就在栈里面！！
现在栈里面是1 5 6，那么6左边第一个比自己小的不就是那个5嘛！

2进来后，6找到了右边比自己小的，6弹出的时候，同时记录右边比自己小的，以及栈顶（也就是左边比自己小的），这样就行了哈哈哈

```python 
class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        n = len(heights)
        stack = [0]#依旧是记录下标
        #先构建右边第一个比自己小的下标在哪
        ans = 0
        for i in range(1,n):
            while stack and heights[stack[-1]] > heights[i]:#当栈不为空，并且来了一个小的
                #挤走一个-1位置的，记录
                temp_height = heights[stack.pop()]#挤走的下标的高度
                temp_left = stack[-1] if stack else -1#挤走的左边的第一个小的下标,如果stack空说明没有，直接就是-1，待会减的时候+1
                temp_right = i#右边的子一个小的下标
                ans = max(ans ,temp_height*(temp_right-temp_left-1))
            #结束完后，记得把i放入栈
            stack.append(i)
        return ans 
```

有几个问题，首先是宽度的计算，应该多减一个，因为左边界和有边界都不参与计算啊，

此外，如果遇到一个递增的数组，比如2 4，那么后面没有比自己小的啊，最后栈不是空的，

所以我们要加一个清栈的结束,相当于是拿n位置0元素去清掉


```python 
class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        n = len(heights)
        stack = [0]#依旧是记录下标
        #先构建右边第一个比自己小的下标在哪
        ans = 0
        for i in range(1,n):
            while stack and heights[stack[-1]] > heights[i]:#当栈不为空，并且来了一个小的
                #挤走一个-1位置的，记录
                temp_height = heights[stack.pop()]#挤走的下标的高度
                temp_left = stack[-1] if stack else -1#挤走的左边的第一个小的下标,如果stack空说明没有，直接就是-1，待会减的时候+1
                temp_right = i#右边的子一个小的下标
                ans = max(ans ,temp_height*(temp_right-temp_left-1))
            #结束完后，记得把i放入栈
            stack.append(i)
        #清栈,相当于是用一个0来去清，把大家都挤走算
        while stack:
            temp_height = heights[stack.pop()]#挤走的下标的高度
            temp_left = stack[-1] if stack else -1
            temp_right = n#相当于是算的最后的，n位置有一个高度的是0的
            ans = max(ans ,temp_height*(temp_right-temp_left-1))

        return ans 


```
