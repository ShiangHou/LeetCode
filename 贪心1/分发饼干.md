# 分发饼干

## 初步思路
这题说的是，给我一个数组，是孩子最低能够满足的胃口

然后又给我一个数组，是饼干的大小

问我最多能使几个孩子满足

这题感觉就是，首先我先排序，

然后，我拿饼干从小往大遍历孩子，如果遇到了比饼干大小低的，那么就直接加1，即喂饱了这个孩子，然后再拿下一个饼干开始往下就行

如果发现孩子胃口比自己大，由于是排序的，既然前面的都没满足，那么后面的也无法满足，说明这个饼干没有用（因为一个孩子只能吃一块饼干），所以此时从下一个饼干遍历就行，然后从当前这个没成功的孩子开始就行


```python 
    def findContentChildren(self, g: List[int], s: List[int]) -> int:
        g.sort()
        s.sort()
        ans = 0
        #先用饼干
        i = j = 0#i是孩子，j是饼干
        while i < len(g) and j < len(s):
            if s[j] >= g[i]:#如果饼干可以满足孩子
                ans += 1#答案+1
                i += 1
                j += 1#然后往下找下一个饼干和孩子
            else:#如果满足不了，就换一个更大的饼干试试
                j += 1
        return ans
```
