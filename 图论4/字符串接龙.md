## 初步思路

题目都没看懂

就是说，我现在有两个目标，start 和end，我最终需要把字符串从start改到end

但是我不能随便改，改的话

1:一次只能改一个字母

2:不能随便改，只能从一个列表里面选择可以变成的样子

换句话说，我需要找到一条路径，start - 列表里面来回变- end，并且每个都只能变个字母

比如说start 和end 是abc def

列表是 efc dbc ebc dec dfc yhn

那么我首先可以从abc变成dbc，dbc在表里，然后再变成dec，最后出来，变成def，这样变了3次（经过了四个点），最后输出4

那么这个可以看成是一个图，我们可以把表里的字符串都建图出来，

建图，如果两个字符串之间是就差一个的话，那么就链接在一起，然后这里可以建一个双向图，节点是字符串本身

然后start 是1，end是n，加上表构建图，去找一个1到n的最短路径就行


可以写一个函数，判断两个字符串是不是只差一个，用邻接列表


但是这里可以用set，因为hash是O1，就是说如果主动变身，比如abc，首先a变成a到z，然后看一下是不是在表里，是的话直接加入边，然后继续，找到所有的边，然后继续往下

那么如果用set的话可以用广搜，遍历到了就往队列里面加

因为这里问的是最短的路径，如果我用广搜的话，一层一层来，当第一次遇到end的时候就一定是最短的


```python 
from collections import deque


def bfs(graph_set,begin,end):#从begin到end的最短路径
    #思路是队列里出来一个元素，然后把每一个位置的每一个字母遍历，看看在不在set里面，在的话加入队列

    dq = deque()
    #先把start加进去
    dq.append(begin)
    ans = 1#记录几个节点，start也是一个节点，所以从1开始

    while dq:
    #为了统计节点数，这个弹出+加入应该持续len(q)次，相当于一层，
    #但是为了防止弹出后加进去，需要先获取一下长度
    #所以是while里面套一个for循环，当qd不是空的时候，然后更新长度，然后for循环开始弄长度
        n = len(dq)
        for _ in range(n):
            #弹出去一个
            temp = dq.popleft()

            #如果弹出的在set里面，就删除它，相当于这个处理过了，防止重复处理


            #把弹出去的所有位置的都换一遍
            #python不能原地修改字符串，我们可以直接切片
            for i in range(len(temp)):#遍历每一个位置
                for j in range(26):#遍历每一个字母
                    new_char = chr(ord('a')+j)#就是ascii码，用a位置的+j就是第几个字符,new_char就是 a b c等字符
                    

                    #如果字母和原来的一样，就继续
                    if new_char == temp[i]:
                        continue 


                    new_word = temp[:i]+new_char+temp[i+1:]#切片拼装起来

                    #如果新的字母是答案，说明已经找到了,直接返回统计的数量
                    if new_word == end:
                        return ans+1#+1是因为最后一个end也算


                    #如果新的字母在表里面，就加入到dq
                    elif new_word in graph_set:
                        dq.append(new_word)
                        #在set里面删除它，相当于这个处理过了，防止重复处理
                        graph_set.remove(new_word)
        
        #每一层结束后，ans再加1，相当于走完了一层
        ans += 1
    #如果中间没有抵达end的return，说明没找到，return 0
    return 0

def main():
    n = int(input())

    begin, end = map(str,input().split())

    graph_set = set()

    for _ in range(n):
        graph_set.add(str(input()))
    dq = deque()

    print(bfs(graph_set,begin,end))

if __name__ == "__main__":
    main()


```
虽然写了很久，但还是两遍过，有一些细节

首先是图的构建，其实不用建图，用主要的遍历比建图更快

然后是要用bfs，因为找的是最短路径

bfs的时候的ans应该是一层一层的，就是左神说的 重复len 次，即一个while加一个for循环
