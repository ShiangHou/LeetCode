## 初步思路
写一个算法判断是不是快乐数

所谓快乐数就是，将所有的位平方相加，一直进行下去，直到最终变为1

初步看起来一点思路都没有啊，

注意到提示是1 <= n <= 2^31 - 1

那么一个奇思妙想是，直接构建一个长度是2^31 - 1的数组当成hash

对于一个n，如果小于10，直接输出false，如果大于10，先记录一下在hash中的位置，然后转成字符串开始遍历，计算新的n，进入下一个循环就行

首先一个错误的想法是，直接用2^31 - 1的数组来去弄，结果发现直接溢出内存，

```python 
def isHappy(self, n: int) -> bool:
        hash_nums = [0]*((1 << 31) - 1)
        while hash_nums[n] != 1:
            hash_nums[n] = 1 #记录一下新的n
            if n < 10:
                return False
            n = 0
            for i in str(n):
                n+=int(i)**2
            if hash_nums[n] == 1:#出现循环
                return False
            if n == 1:
                return True

```


正确的应该是用一个集合吧

## python中集合set的用法
在python中，set是一个标准的去重hash，查找的复杂度就是O(1),

一些set的基本用法

```python 
myset = set()#创立新的集合
myset.add(4)#添加元素
myset.remove(4)#删除元素
if 2 in myset:
    pass
```
知道了集合的用法后,就可以做着题了

## 第一版思路

```python 
def isHappy(self, n: int) -> bool:
    n_set = set()
    while n not in n_set:
        n_set.add(n)
        new_n = 0
        for i in str(n):
            new_n += int(i)**2
        if new_n == 1:
            return True
        if new_n in n_set:
            return False#出现循环
        n = new_n

```
