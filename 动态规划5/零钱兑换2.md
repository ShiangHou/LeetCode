# 零钱兑换2

给了一个amount，要从coin数组中取硬币，使和等于amount

首先暴力递归就是，我去遍历每一个i，然后分为取和不取，因为这题可以无限用，所以不用的话直接跳到下一个就是，但是如果用的话，就是继续递归fi

转换成dp的话，一个二维dp，i和amount，然后填表就行


注意，所有的背包问题，不使用当前的硬币，一直是j，因为不使用当前的意味着需要从之前的组合中凑出j，

而如果使用了当前的硬币，那么就说明，留给之前的变少了，即j- amount

```python 
class Solution:
    def change(self, amount: int, coins: List[int]) -> int:
        n = len(coins)
        dp = [ [0]*(amount+1)for _ in range(n+1)]
        #初始化
        for i in range(n+1):
            dp[i][0] = 1#凑成数量为0的就是一个都不取
        for j in range(amount+1):
            dp[0][j] = 0#0个硬币无法凑成任何一个，除了0-0
        dp[0][0] = 1
        for i in range(1,n+1):
            for j in range(1,amount+1):
                if j -coins[i-1] >= 0:
                    dp[i][j] =dp[i-1][j]+ dp[i][j-coins[i-1]]
                else:
                    dp[i][j] =dp[i-1][j]
        return dp[n][amount]

```
