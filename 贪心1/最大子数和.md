# 最大子数和
力扣 53

## 思路

题目问的是，给一个数组，找到连续的最大和，

即要求找到的片段是数组中是连续的，而且要和最大

## 提示

厚米提示，贪心算法是这样的，我不断拿着数字往包里加，如果包的总和是正的，那么下一个只要拿完后还是正的（无论下一个是正负），那我就都要拿，如果拿过后发现包变负了，果断舍弃所有的数从头再来

同时维护一个max就行

此外，这个max设置的很有讲究，比如 -3 -5 -2 ，那么其实不能每次都清零，最后是-2，所以max最初应该是包的量

```python 
    def maxSubArray(self, nums: List[int]) -> int:
        _max = float('-inf')
        temp = 0
        for i in range(len(nums)):
            #先记录当前成绩
            temp += nums[i]
            #看看有没有破纪录
            _max = max(temp,_max)
            #然后判断一下
            if temp <0:
                temp = 0
        return _max
```
