## 初步思路

题目说是给了我们一个整数，问有没有一条路径，从根节点开始往下找，能够等于这个目标值的

再注意一下，说是从根节点到叶子节点，也就是说一定要走到底才行，不走到底就不返回

感觉就是一个递归。递归好好考虑一下怎么写

构造一个递归函数f，入参是node，sum两个，
首先判断node的左右都有没有了，如果没有了就直接算一下 sum+node.val，如果等于的话，就返回True就行

如果有，就继续递归，f(node.left,sum+node.val)

最后调用这个递归，如果是True就返回True，如果没返回就false


```python 
    def hasPathSum(self, root: Optional[TreeNode], targetSum: int) -> bool:
        if not root:
            return False
        self.sum = targetSum
        self.flag = False
        self.f(root,0)

        return self.flag
        

    def f(self,node,count):
        if not node.left and not node.right:
            if count + node.val == self.sum:
                self.flag = True
        if node.left:
            self.f(node.left,count+node.val)
        if node.right:
            self.f(node.right,count+node.val)

```

## 优化

最好还是不要用全局变量，好厚米给了我一个小巧思，我们只需要在遍历的时候不断减去node.val，如果到最后target正好是node.val，那么就返回true就行，不是就返回false

```python 
def hasPathSum(self, root: Optional[TreeNode], targetSum: int) -> bool:
    if not root:
        return False
    if not root.left and not root.right:
        return targetSum == root.val
    return self.hasPathSum(root.left,targetSum-root.val) or self.hasPathSum(root.right,targetSum-root.val)
```
