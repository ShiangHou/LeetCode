## 初步思路

题目要求找到这个字符串的全部符合回文串的子串，

那么全部分割成单个的肯定是可以的

首先的想法就是先把所有的子串都找出来，然后把不符合的斩杀就行，

当然是不可以的。

嗷嗷嗷，注意到了一个华点，要求是分割字符串，即我把字符串切开，切开后的所有的内容都应该是回文的

所以刚才那个想法其实就不太对

换句话说，这是一个递归回溯的过程，我先切，遇到一个回文就切，切到最后如果都是回文就返回，不都是回文就说明之前的一个切的不对，返回去重新切

梳理一下思路，这个过程有两个

一个是对于原始字符串的分割点

一个是分割后，判断现在分割点左边的是不是回文，如果是的话，就继续在右边的递归去做，
做完后返回到最开始的原始分割点，然后原始分割点往后移动一步

所以是一个for循环 里面嵌套递归，for循环就是从0-n切每一刀，然后for循环中间嵌套递归



```python 
    def partition(self, s: str) -> List[List[str]]:
        self.ans = []
        self.f(s,0,[])
        return self.ans
    
    def f(self,s,start,path):#start是从左边开始往后的切分的位置
        if start == len(s):#如果切到顶了，就结束
            self.ans.append(path.copy())
            return 
        #对于当前这层，先每一步每一步切
        for i in range(start,len(s)):
            #切，切完后，左边是start到i，右边是i+1到len
            temp = s[start:i+1]
            #判断一下是不是回文
            if temp == temp[::-1]:#是回文，往后递归
                path.append(temp)
                self.f(s,i+1,path)
                #记得取出来
                path.pop()
            else:#不是，直接继续循环就行
                continue
```
