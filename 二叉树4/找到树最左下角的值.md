## 初步思路

最左下角的值首先想到的是层序遍历，我把层序遍历最后的[-1][0]直接弹出来不久行了嘛，也就是层序遍历的最后一层的第一个，就是最左边的啊

```python 
    def findBottomLeftValue(self, root: Optional[TreeNode]) -> int:
        dq = deque()
        ans = []
        dq.append(root)
        while dq:
            size = len(dq)
            temp_ans = []
            for _ in range(size):
                temp = dq.popleft()
                temp_ans.append(temp.val)
                if temp.left:
                    dq.append(temp.left)
                if temp.right:
                    dq.append(temp.right)
            ans.append(temp_ans)
        return ans[-1][0]

```
