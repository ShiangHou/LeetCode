# 电话号码的字母组合
力扣17
## 思路
题目说是要把电话号码的数字进行映射，就是九宫格打字，把2-9打出来的所有字母组合都存进去

这题首先是要把2-9的全部都存到一个hash里面，当然也可以是一个数组，然后用下标就行

梳理一下流程，这里要想象是一个多叉树

比如要23

对于第一个节点2，选择要a/要b/要c，然后继续往下，对于3，选择要d/要e/要f，如果最后长度等于23的长度，保存一下就行

```python 
    def letterCombinations(self, digits: str) -> List[str]:
        self.ans = []
        self.digits = digits
        self.num_al = [
            "",
            "",
            "abc",
            "def",
            "ghi",
            "jkl",
            "mno",
            "pqrs",
            "tuv",
            "wxyz"
        ]
        self.f(0,"")
        return self.ans
    def f(self,i,path):#i是digits的位置，即i为0，走到2或者3
        if len(path) == len(self.digits):#收集的path等于数字的长度了，满足，就返回
            self.ans.append(path)
            return
        if i > len(self.digits) and len(path) != len(self.digits):#i越界了，但还没搜集完，直接返回
            return 
        for j in self.num_al[int(self.digits[i])]:#从digits的0位置找到数字，然后再到hash中去找
            self.f(i+1,path+j)#往下走
        return path

```

注意，python的字符串不需要pop，因为python中对于+完后会在内存里面生成一个新的，让回溯回来后不会改变当前的
