## 初步思路

题目说，要在两个数组之间划线，
比如

1 2 4

1 4 2

其中1 和1 可以连在一起，2后后面那个2可以连在一起，但是第一个4和前面那个四无法连在一起

额，这个不是就是，emmm，递增的，公共的，子序列嘛

那么我们依旧是用一个二维的动态规划

因为是公共的，所以需要判断nums1[i]和nums[j],如果一样的话才能

回想我们在最长递增子序列里面，是需要遍历前面的，但其实这里我们不用，因为只需要递推前面的就行

哎哎哎，这个不需要递增啊，刚才那个例子只是2在前面而已，没有要求递增

那就是和最长公共子序列一样，如果等于的话就是 i-1 j-1 +1，如果不等的话就是 max i-1 j和 i j-1

```python 
class Solution:
    def maxUncrossedLines(self, nums1: List[int], nums2: List[int]) -> int:
        n = len(nums1)
        m = len(nums2)
        dp = [[0]*(n+1) for _ in range(m+1)]
        for i in range(1,m+1):
            for j in range(1,n+1):
                if nums1[j-1] == nums2[i-1]:
                    dp[i][j] = dp[i-1][j-1]+1
                else:
                    dp[i][j] = max(dp[i-1][j],dp[i][j-1])
        return dp[m][n]


```
