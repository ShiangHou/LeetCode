就像刚才想的，刚才想的**我们需要判断的是，左的左和右的右；左的右和右的左是不是一样的**就行，刚才我认为是不是还没考虑周全，其实逻辑上是周全的，但是标的不周全，我刚才只考虑了node，我们把这个升级到**子树**概念就行，这样的话就可以了

```python 
def isSymmetric(self, root: Optional[TreeNode]) -> bool:
    if not root:
        return 
    return self.check(root.left,root.right)

def check(self,left_node,right_node):
    if not left_node and not right_node:
        return True
    if not left_node and right_node:
        return False
    if left_node and not right_node:
        return False
    if left_node.val != right_node.val:
        return False
    return self.check(left_node.left,right_node.right) and self.check(left_node.right,right_node.left)
```
