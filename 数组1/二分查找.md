# 二分查找 
题目链接https://leetcode.cn/problems/binary-search/

本题是一道比较基础的题目，重点是要学会边界的替换

当while循环条件是l<=r时，即左闭右闭，所以l和r的位置都有可能是这个target的值，因此初始化时候l要是0，r要是n-1
```python
def search(self, nums: List[int], target: int) -> int:
        n = len(nums)
        l,r = 0,n-1
        while l <= r:
            m = (l+r)//2
            if nums[m] == target:
                return m
            else:
                if nums[m] < target:
                    l = m+1
                else:
                    r = m-1
        return -1
```

还有一种写法，如果我们的循环条件时l < r,即左闭右开，此时r位置就不可能是target的值了，

此时有三处变动
1：循环条件时l < r

2：初始化的时候，r是n

3：缩小范围的时候，由于r要是开区间，所以和开始的n一样，往右走一位就行
```python
def search(self, nums: List[int], target: int) -> int:
        n = len(nums)
        l,r = 0,n
        while l < r:
            m = (l+r)//2
            if nums[m] == target:
                return m
            else:
                if nums[m] < target:
                    l = m+1
                else:
                    r = m
        return -1
```
