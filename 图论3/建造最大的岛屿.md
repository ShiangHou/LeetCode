## 初步思路

题目说，我现在有一个岛屿，然后我可以任意指定一块0变成1，变成1后，这块岛屿可能会与其他陆地连接在一起，这样就可以形成一块陆地了，

然后问，把哪块水变成陆地可以使得原来的陆地面积最大


emmm，这道题首先问题在于，怎么去找那个水在哪

换句话说，我需要像强化学习那样，衡量出每个水变成陆地的价值

可以这样，首先我先统计出所有的岛屿

然后遍历所有的水，用dfs去检索（假设这个水是陆地），然后得到每个水的价值，也就是面积，最后返回面积最大的一个

现在时间复杂度会爆表，原因在于每次遍历的时候都要深搜，但其实我可以就深搜一次，把所有的陆地以及面积都找到并且存起来，

然后我在遍历水，如果遇到了陆地，直接连带着这片陆地的面积直接返回就行了，不用往下深搜了

那么问题转化为，怎么样在第一次深搜的时候在graph上标记出这块陆地的记号，以及对应的面积是多少

这个其实简单，我在递归里面加上一个编号，假设是mark，在main函数里面，如果遇到了陆地，就带着mark去深搜，搜完后这一片陆地就打上了mark，并且统计完了面积，然后我存起来就行了，然后mark+1，tmep的面积清零开始下一个就行

def dfs(graph,i,j,visited,mark,direction,n,m):
    if graph[i][j] == 0 or visited[i][j] == True:#如果遇到水或者已经遍历过了
        return 0
    #先把当前的计入到visted
    visited[i][j] = True

    #记录一下当前的mark
    graph[i][j] = mark

    area = 1#自己有一个面积
    for x,y in direction:
        next_i,next_j= i+x, j+y
        if next_i<0 or next_i>n-1 or next_j < 0 or next_j > m-1:#越界了继续
            continue
        else:#直接深搜，加上这个
            area += dfs(graph,next_i,next_j,visited,mark,direction,n,m)


    #最后往上返回面积
    return area
