## 初步思路

复述一下题目，题目说是给了一个数组，从中随机的取两个数，如果一个数大于另一个数，(比如y大于x)，那么就让另一个数等于0，然后让y = y-x，就这样，最后可能只会剩下一个石头，问这个石头最小是多重

看了一下例子，这个随机取还是有讲究的，因为如果是随便取的话，最后碰完剩下的可能是最大的

emmmm怎么转化为背包问题捏

把石头分成两堆，使得两堆的重量尽可能的相等（尽可能靠近sum/2），这样子，两堆石头碰的时候就会有最小的差值了

```python 
class Solution:
    def lastStoneWeightII(self, stones: List[int]) -> int:
        target = sum(stones) // 2
        n = len(stones)
        dp = [0]*(n+1)#这个的含义是，前i个石头能够最大凑成多少（并且需要小于等于
        #初始状态，dp0就是前0个能凑成多少，当然是0
        dp[0] = 0
        for i in range(1,n+1):
            #递推公式是，取了i，和没要i，然后要看一下有没有超过target，没有超过的话取最大，超过的话就只能去没超过的，
            if dp[i-1]+stones[i-1]<= target:#如果加了之后没有超过，可以考虑加一下
                dp[i] = max(dp[i-1],dp[i-1]+stones[i-1])
            else:
                dp[i] = dp[i-1]
        #最后算一下和target的差值
        return dp[n] - target
```
这里是有问题的，首先是dp定义的问题，不能这么定义，这样会忽略掉一些组合

比如 3 4 5，target是6

最开始的时候是3 ，然后我们看4，因为4+3大于6，此时还是3，但是其实4是离6更近的，这没有被察觉到


所以我们应该改一下定义，即每一个阶段所有可能的合法子集，

所以说我们应该多加一个变量 j，即限制一下总重量，然后找到不超过改重量的情况下的最大值


```python 
class Solution:
    def lastStoneWeightII(self, stones: List[int]) -> int:
        target = sum(stones) // 2
        n = len(stones)      
        dp = [ [0]*(target+1) for _ in range(n+1)]#这里的含义就是，前i个，大小不超过j，的最大的能组合的重量是多少
        # for j in range(target+1):
        #     dp[0][j]  = 0#前0个能够组合的重量，没有取，所以也是0
        
        # for i in range(n+1):#前0个能够组合的重量，没有取，所以也是0
        #     dp[i][0] = 0
        #那其实可以不用初始化

        for i in range(1,n+1):
            for j in range(1,target+1):
                    #这里的递推是，用了i-1的和没用的，用了i-1的话，那么j那里就是要减去就行
                    #减去要看有没有到0
                    if j-stones[i-1] >=0:
                        dp[i][j] = max(dp[i-1][j],dp[i-1][j-stones[i-1]]+stones[i-1])
                    else:
                        dp[i][j] = dp[i-1][j]
        dp_max = 0
        for i in range(1,n+1):
            for j in range(1,target+1):
                dp_max = max(dp[i][j],dp_max)
        return sum(stones) - 2*dp_max
```

有几个问题注意，首先是递推，dp[i-1][j-stones[i-1]]+stones[i-1] 需要加上stonepi-1才行，因为这里的

dp[i-1][j-stones[i-1]]的意思是，重量最大是j-stones[i-1]的重量最大是多少，

首先，dp[i-1][j]的意思是，只用前i个就凑到了j重量的

dp[i-1][j-stones[i-1]]的意思是，用前i个就凑到了j-stones[i-1]（即不含当前的）重量的，那么我现在是dp[i][j]，

既然决定放入第 $i$ 个石头，该石头必然会严格占据等同于 stones[i-1] 的物理容量。
为了确保放入该石头后总重量不超标，前 $i-1$ 个石头所能使用的物理容量上限，必须被强制缩减为 $j - stones[i-1]$

而决定放入当前石头后，最终的总重量应当等于历史阶段合法积累的重量加上当前石头的重量。
