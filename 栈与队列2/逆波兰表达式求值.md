## 初步思路

读一遍题目，所谓的逆波兰表达式就是，比如

tokens = ["2","1","+","3","*"]

就是（2+1）*3

观察了题目中的几个，发现是有规律的

首先，当出现运算符号的时候，需要把之前加入的数字运算一下，然后再放回去就行

也就是说，我准备一个栈就可以

首先，遇到数字，就压入栈，

如果遇到一个符号，比如+，就弹出2次栈内的元素，一个a，一个b，然后执行a+b，然后再压会栈中就行，最后栈里面应该就只剩一个数字了，直接返回就行

有几个问题，比如说我int(i)后如果i是字符怎么办（即怎么去判断字符串里面是不是数字还是+-*/

这个好解决，先判断字符串是不是+-*/，然后一个else，这样剩下的就都是数字来

还有一个问题，题目中的**向0截断**是什么意思

意思就是说，除的话，将0后面的所有小数都删除，所以说//这个就不行，因为这是向下取整，遇到负数就不行，可以直接int(a/b)

```python 
def evalRPN(self, tokens: List[str]) -> int:
        stack = []
        for i in tokens:
            if i == "+":
                b = int(stack.pop())
                a = int(stack.pop())
                stack.append(str(a+b))
            elif i == "-":
                b = int(stack.pop())
                a = int(stack.pop())
                stack.append(str(a-b))
            elif i == "*":
                b = int(stack.pop())
                a = int(stack.pop())
                stack.append(str(a*b))
            elif i == "/":
                b = int(stack.pop())
                a = int(stack.pop())
                stack.append(str(int(a/b)))
            else:
                stack.append(i)
        return int(stack[-1])

```
