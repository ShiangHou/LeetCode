## 初步思路

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。 

每次你可以爬至多m (1 <= m < n)个台阶。你有多少种不同的方法可以爬到楼顶呢？ 

注意：给定 n 是一个正整数。

这个题目很简单，就是一个完全背包

我们有1-m的可以重复用的物品，问怎么最后可以到达n,背包的

```python 
n,m = map(int,input().split())

def f(n,m):
    dp = [0]*(n+1)
    dp[0] = 1 #就是说几种方法可以到达0层，这里就是不动，所以是1
    for i in range(1,n+1):#注意从1开始遍历，不然会刷掉0
        for j in range(m+1):
            if i-j >=0:
                dp[i] +=dp[i-j]
    return dp[n]

print(f(n,m))


```
