##初步思路
题目要求构建最大的二叉树

所谓最大的二叉树，就是指，在一个数组中找到最大值，然后对于这个数组，左边子数组继续递归，右边子数组也是继续递归找到右边的最大值继续，

那么首先就是要写一个函数来去计算一个数组的最大值



```python 
    def constructMaximumBinaryTree(self, nums: List[int]) -> Optional[TreeNode]:
        if not nums:
            return 
        n = len(nums)
        max_num,max_id = self.FindMax(nums)
        node = TreeNode(max_num)
        if max_id - 0 >= 0:
            node.left = self.constructMaximumBinaryTree(nums[0:max_id])
        if len(nums)-1-max_id >=0 :
            node.right = self.constructMaximumBinaryTree(nums[max_id+1:n])
        return node                  
    

    def FindMax(self,nums:List[int]) -> int:
        ans = float('-inf')
        ans_id = -1
        for i in range(len(nums)):
            if nums[i] > ans:
                ans = nums[i]
                ans_id = i
        return ans,ans_id



```
