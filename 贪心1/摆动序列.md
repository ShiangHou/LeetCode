## 思路
首先摆动序列是指，两个数之间的差值（后减前）是一正一负交替出现的，

题目说，给我们一个数组，找到这个数组中最长的摆动子序列

子序列可以是跳着来的

首先摆动序列好理解，现在问题是，怎么在数组中找到符合要求的摆动子序列啊，而且还是最长的，感觉毫无头绪

## 提示

厚米提示的非常到位

首先，我们把数组想象成股票图，来来回回起伏

我们要找的是什么，我们要找“起伏起伏起伏起伏”这种

这就意味着，比如 1 2 3 4 5 4 3 2 1 6这一串数字，从1到5 都是起，从5到1都是伏，所以中间的这些东西可以全都不要，我们只统计所有的峰谷和波底就行

有个问题，对于头和尾需要算在峰谷和波底嘛，

我觉得是算的，因为，比如对于0位置，下面的有可能是往下走，有可能是往上走，但不管怎么走，自己仍是

换句话说，如果是1 1 1 2 4 5 3 这种，那么我们就要从开始变化的那一个启动就行

同时，怎么记录变化呢，我们可以用pre_diff和cur_diff来去统计


```python 


    def wiggleMaxLength(self, nums: List[int]) -> int:
        ans = 0
        cur_diff = 0
        pre_diff = 0
        for i in range(len(nums)-1):
            cur_diff = nums[i+1]- nums[i]
            #如果坡度改变了符号
            if (cur_diff >0 and pre_diff <= 0) or (cur_diff <0 and pre_diff >= 0):
                ans += 1
                #找到极值点后才能更新pre
                pre_diff = cur_diff
        return ans+1


```

注意，找到极值点后才能更新pre
