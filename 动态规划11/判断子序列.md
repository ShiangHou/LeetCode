判断的是s是不是t的子串

```python 
class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        m = len(s)
        n = len(t)
        dp = [[0]*(n+1) for _ in range(m+1)]
        
        for i in range(1,m+1):
            for j in range(1,n+1):
                if s[i-1] == t[j-1]:
                    dp[i][j] = dp[i-1][j-1]+1
                else:
                    dp[i][j]= max(dp[i-1][j],dp[i][j-1])
                
        return dp[m][n] == len(s)
```

```python 
class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        m = len(s)
        n = len(t)
        dp = [[0]*(n+1) for _ in range(m+1)]
        
        for i in range(1,m+1):
            for j in range(1,n+1):
                if s[i-1] == t[j-1]:
                    dp[i][j] = dp[i-1][j-1]+1
                else:
                    dp[i][j]= dp[i][j-1]
                
        return dp[m][n] == len(s)
```

注意，s是短的，m是短的，i是短的，，那么这里的意思就是，用长的j-1的看看能不能拼出
