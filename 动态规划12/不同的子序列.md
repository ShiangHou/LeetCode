## 初步思路

题目要求我们去判断两个字符，s和t，返回在s中t出现的个数

就是说，s中可能出现了很多的t，然后呢组成t的方式有不同，

emmm，我们之前在做 判断子序列的时候，我的定义的dp是出现的这个长度是多少，然后这里是问的次数，思路感觉还是和之前一样，我们用一个二维dp来去做这些，就定义为以i和j为结尾的，能够组成的次数吧

如果结尾不一样，说明本次俩结尾没有组成，那就用长的那个i-1之前的看看有没有组成，

如果结尾一样，那么就是继承i-1和j-1的组成种类然后+1就行

但是这个初始化感觉不能全是0了，就是说，需要看那个00位置是不是一样的

厚米提示

首先是初始化，一个是用空拼出原来的，自然就是只有一种，即全删了，一个是所有的拼成空，自然什么都拼不成，就是1

此外，当我们发现i和j是相等的时候，有两种选择，一种是用i，那么就是继承i-1和j-1，一种是不用i，那么直接继承i-1就行，最后两个去max

```python 
class Solution:
    def numDistinct(self, s: str, t: str) -> int:
        #s长t短
        m = len(s)
        n = len(t)
        dp = [[0]*(n+1) for _ in range(m+1)]
        #初始化
        for i in range(m+1):#t是空，那么s就是全部删除这一种
            dp[i][0] = 1
        # for j in range(n+1):#s是空，那么t就租不成
        #     dp[0][j] = 0

        #上面的别写了，逻辑是对的，但是这样会吧00为 1给覆盖掉
        for i in range(1,m+1):
            for j in range(1,n+1):
                if s[i-1] == t[j-1]:
                    dp[i][j] = dp[i-1][j-1]+ dp[i-1][j]#注意这里是加，因为是所有的种类组合
                else:
                    dp[i][j] = dp[i-1][j]#如果不等，直接看前面的就行
        return dp[m][n]
```
