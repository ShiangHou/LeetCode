## 初步思路
首先题目给了一个数组，一个target，要求找到所有用这个数组里面数字和等于target的所有组合

题目说是同一个数字可以无限用

思路感觉依旧是回溯递归，脑子里是二叉树，对于第一个，分叉用还是不用，

但是这样说是一个数字可以无限用，

先试一下思路是和之前的一样吧


```python 
def combinationSum(self, candidates: List[int], target: int) -> List[List[int]]:
    self.ans = []
    self.f(candidates,target,0,[])
    return self.ans

def f(self,nums,target,i,path):#nums是需要遍历的数组，target是组合的数，i是现在需要到nums的第几个下面了,path是已经搜集的路径
    if target < 0:#上一步的时候把target减成负了，这个组合直接遗弃
        return 
    if target == 0:#正好等于，保存答案
        self.ans.append(path.copy())
        return path
    if i >= len(nums):#如果i遍历过一遍了，并且没有命中上面的等于
        return 
    #如果target是大于，那么继续分情况遍历
    #用i位置
    path.append(nums[i])
    self.f(nums,target-nums[i],i,path)
    path.pop()
    #跳过了i位置
    self.f(nums,target,i+1,path)
    return path
```
