## 最后思路

刚才说的思路是对的，我们写的时候注意一些小细节就行

有一个最终的result，然后递归函数里面是path，path应该是一个字符串

首先先写终止条件，当node的左和右都是空的时候，此时说明到头了，直接把node.val拼在path里面返回就行，最后拼在result里面

当有左的时候，记录node.val -> 拼在字符串里面，然后继续调用这个就行

厚米和我说，python中的字符串属于不可变类型，每次往下传递的时候都是一个新的字符串，所以不用回溯。但是C++和java需要回溯

```python
def binaryTreePaths(self, root: Optional[TreeNode]) -> List[str]:
    if not root:
        return [""]
    self.result = []
    self.DFS(root,"")
    return self.result

def DFS(self,node,path):
    if not node.left and not node.right:
        path = path + str(node.val)
        self.result.append(path)
    path = path + str(node.val)+ "->"
    if node.left:
        self.DFS(node.left,path)
    if node.right:
        self.DFS(node.right,path)
```

其实整体还是非常的简单
