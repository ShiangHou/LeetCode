## 初步思路

这题就是一个树上的打家劫舍，数形dp

看了左神的讲解

首先，我们需要定义两个全局变量，一个叫做yes，一个叫做no，yes的意思是说，取当前这个节点的val，以及所有的子树所能得到的和，no的意识是，不取当前这颗树所能造成的和

当处在某个节点的时候，我们有两种情况，假设节点是a

a_yes是，取了a，那么就是 val

a_no的意思是没取a，那么它的值需要看左右子树

都要看左右子树，a_yes的时候，a.left和a.right 都不能取，这不就是两个的no嘛，即 a.left-no 和 a.right -no，因此a_yes

a_yes = val+ a.left_no + a.right_no

那么a_no呢，因为这俩子树都是取和不取都可以，因此左子树最大是max(a.left_yes,a.left_no)，那么同理，右子树的最大是max(a.right_yes,a.left_no)

写一个递归就行


```python 
class Solution:
    def rob(self, root: Optional[TreeNode]) -> int:
        self.yes = 0
        self.no = 0
        self.f(root)
        return max(self.yes,self.no)

    def f(self,node):
        if node == None:
            self.yes = 0
            self.no = 0
        else:
            #先看用了的
            y = node.val

            #没用的
            n = 0
            self.f(node.left)#调用后会更新全局变量yes和no，此时的yes和no就是左子树的yes和no
            #用了的话
            y += self.no#左边的no

            #没用的话,是左的yes和no的最大
            n += max(self.yes,self.no)
            #继续，刷一下，把yes和no刷到是右树de

            self.f(node.right)
            y += self.no
            n+=max(self.yes,self.no)

            #最后记得刷当前节点的yes和no
            self.yes = y
            self.no = n

```
