## 初步思路
这题之前用贪心的做法是，遍历到一个数，加上去temp，更新max，看一下这个是不是负的，如果是负的话直接temp弄成-

用dp的话，就是定义前i个数的最大和就行，思想感觉差不多

注意，这里的dpi应该是以i结尾的子数组的最大和

当我们遍历到i的时候，继续分为用自己和不用自己


如果吧自己加上，如果加上的话是负的，那么直接以自己作为起点就行
换句话说，如果加上自己后，都没有自己大的话，那么直接以自己作为起点就行哈哈哈

也就是nums[i]+dp[i-1] ,dp[i] 取max

然后整体取一个最大就行

```python 
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        n = len(nums)
        dp = [0]*(n)
        dp[0] = nums[0]
        for i in range(1,n):
            dp[i] = max(dp[i-1]+nums[i],nums[i])
        return max(dp)
```

这里要区分清楚数组的下标i还是i-1
