## 初步思路
这里说，word1变成word2，可以增删改查

emmmm，那就定义dp i和dpj是可以改变的最小的次数就行

然后初始化，
dp[i][0]就是用0去构建0，那么就是一直删除行，来一个删一个

同理，dp[0][j]也是，来一个加一个

递推公式的话，如果发现ij位置是相等的，那么就直接继承上一个的

如果不相等的话，emmm，是不是要看增删改这三种情况

增的话就是加上一个，强行使得这个位置是匹配的，那么就需要继承j-1了（因为i是一样的）那么操作就是要+1

删的话，就是利用i-1前面的来去拼j

改的话就是直接强行匹配，就和ij位置相等的是一样的

然后这三种是操作都要+1

```python 
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m = len(word1)
        n = len(word2)
        dp = [[0]*(n+1) for _ in range(m+1)]
        #初始化
        for i in range(m+1):
            dp[i][0] = i
        for j in range(n+1):
            dp[0][j] = j
        
        #开始
        for i in range(1,m+1):
            for j in range(1,n+1):
                if word1[i-1] == word2[j-1]:
                    dp[i][j] = dp[i-1][j-1]#如果是相等的，那么直接继承
                else:
                    dp[i][j] = min(dp[i][j-1],dp[i-1][j],dp[i-1][j-1])+1#分别对应增，删，改
        return dp[m][n]
```
