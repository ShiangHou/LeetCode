# 完全平方数

力扣279

## 初步思路

题目说，给我们一个数组，问这个数字最少可以由几个完全平方和的数来去组成

比如说13，那么其实可以用9+4来去组成

然后12可以用 4+4+4来去组成

换句话说，我有1 4 9 16等等的数字，我要从这些数字里面任意取，使其和等于num，并且拿的数量最少

这其实就是一个多重背包的问题

那么首先我们需要看一下最大能到多少，其实是sqrtn就行

```python 
class Solution:
    def numSquares(self, n: int) -> int:
        import math
        dp = [float('inf')]*(n+1)
        dp[0] = 0#注意这里是0，因为后面会加1
        for i in range(1,n+1):
            for j in range(int(math.sqrt(n))+1):
                if i>= j*j:
                    dp[i] = min(dp[i],dp[i-j*j]+1)#这里要加1
        return dp[n] 




```
