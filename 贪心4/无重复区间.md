# 无重复区间
力扣435

## 初步思路

题目说是要找到重复的区间，然后最小次数删除掉重复的区间

这个跟气球那一题差不多啊，先排序，排完序后，如果有两个是挨着的，那么必然要删掉一个，删掉小的还是打的呢

删掉大的，然后重新统计范围就行，范围是i+1和i的最小值



```python 
class Solution:
    def eraseOverlapIntervals(self, intervals: List[List[int]]) -> int:
        intervals.sort(key =  lambda x: x[0])
        count = 0
        for i in range(len(intervals)-1):
            if intervals[i+1][0] < intervals[i][1]:
                #直接删掉，
                intervals[i+1][1] = min(intervals[i][1],intervals[i+1][1])
                count += 1
        return count

```

注意，这里删除的应该是两个之间右边界短的那个，这样可以为后面流出更多的空间
