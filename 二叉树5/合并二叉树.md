## 初步思路
就是说，给我两个root，如果都有某一个节点，那么就把值相加就行，如果有一个没有，但另一个有了，就直接覆盖就行

思路就是说，首先给一个主树，另一个辅助它，如果都有，那么主树+辅助，如果一个有一个没有，那么重构，

处理完这个节点，然后递归的处理node.left和node.right就行

```python 
    def mergeTrees(self, root1: Optional[TreeNode], root2: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root1 and not root2:
            return 
        if root1 and root2:
            root1.val += root2.val
        if not root1 and root2:
            root1 = TreeNode(root2.val)
        if root1 and not root2:
            root2 = TreeNode(root1.val)
        root1.left = self.mergeTrees(root1.left,root2.left)
        root1.right = self.mergeTrees(root1.right,root2.right)
        return root1
```

注意写这种递归的时候要用个变量接着它，即**root1.left** = self.mergeTrees(root1.left,root2.left)

## 优化

写的其实不够优雅，当两个node中有一个没有的时候，其实不用去建这颗树，直接return那个不空的就行

```python 
def mergeTrees(self, root1: Optional[TreeNode], root2: Optional[TreeNode]) -> Optional[TreeNode]:
    #修改root1
    if not root1:
        return root2
    if not root2:
        return root1
    root1.val += root2.val
    root1.left = self.mergeTrees(root1.left,root2.left)
    root1.right = self.mergeTrees(root1.right,root2.right)
    return root1

```
