# 移除元素
力扣27 

## 初步想法

这道题首先想到的是交换，即当我遍历到一个为val的数后，我就把它和后面的交换，然后跳过后面的数继续遍历

但是这有一个问题是，交换后跳过的那个数是不是符合val，所以还要继续换

因此第一次手写的想法是，我遍历一遍这个数i，当我找到一个等于val的时候，我要把它放到后面去，所以再嵌套一个while循环，找到这个i+1往后第一个和不是val的，然后和它交换，然后i继续走，这样就把所有val的放到后面了

在这注意几个边界条件，首先时i要遍历到最后一个，其次是注意，在while循环中，如果遍历完发现还是空，并且count还是0的时候，说明全都是val



下面是可以通过的代码


```python

def removeElement(self, nums: List[int], val: int) -> int:
    count = 0
    for i in range(len(nums)):
        if nums[i] == val:#如果当前这个等于val，就往后找一个不是val的和他换就行
            j = i+1
            while j<len(nums):
                if nums[j] != val:
                    nums[i],nums[j] = nums[j],nums[i]
                    count+=1#从后面找到一个不是value的，加1
                    break
                #如果j遍历完发现全都是val，说明没有了，已经做完了，之间返回就行
                if j == len(nums)-1 and nums[j] == val:
                    return count
                if j == len(nums)-1 and nums[j] == val and count == 0:#如果遍历完发现count还是0，说明这个nums全都是val，这时直接返回count，并且把nums弄为空
                    nums = [None]*(len(nums)-1)
                    return count
                else:
                    j+=1

        else:#如果当前这个不等于val，直接加1就行
            count+=1
    return count#最后别忘了返回count数

```
## 优化

分析一下，时间复杂度其实还有，原因在于，我们每次都是从i+1往后遍历的，这样其实消耗掉了很多重复的遍历，那么一个很好的思想是快慢指针

这题的本质就是，找到所有不等于val的值，丢在前面就行，所以说fast指针一遍遍历，遇到不等于val的，和slow的值交换，然后slow++，就行

```python 
def removeElement(self, nums: List[int], val: int) -> int:
        slow = 0
        for i in range(len(nums)):
            if nums[i] != val:
                nums[slow],nums[i] = nums[i],nums[slow]
                slow+=1
        return slow

```
