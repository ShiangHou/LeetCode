## 初步思路

这题说，给了我们一个无向的图，问source节点能不能抵达destination 节点的路径，能的话输出1，不能的话输出-1

这题就直接dfs或者bfs啊，very的简单，但是图论5是并查集，所以我们考虑并查集的写法

首先应该是，我们有n个节点是知道的

然后这n个节点我们直接全部塞到并查集里面

然后如果来了俩人，就让这俩人认老大就行，即union操作

最后有连接的认，在认老大环节就已经是形成帮派的了

判断俩人是不是有路径，即判断俩人是不是一个帮派的，直接is_same就行


```python 
#首先定义一个并查集的类

class UnionSet:
    def __init__(self,n):
        self.father = [i for i in range(n)]
    
    def find(self,x):
        #如果表里的老大是自己，直接返回
        if self.father[x] == x:
            return x
        else:#如果不是，是别人，直接递归找别人的father，并回溯给自己
            self.father[x] = self.find(self.father[x])#这里的self.father[x]就是别人
            #找到自己的father，直接返回
            return self.father[x]
    
    def join(self,x,y):#将两个人连在一起
        #直接将两个人的老大连在一起就行
        father_x = self.find(x)
        father_y = self.find(y)

        #如果两个人的老大不是同一个人，直接让一个人认另一个人当老大
        if father_x != father_y:
            self.father[father_x] = father_y
    
    def is_union(self,x,y):#判断两个人是不是同一个组
        #直接看是不是同一个father就行
        return self.find(x) == self.find(y)

def main():
    n,m = map(int,input().split())
    #直接把n全部塞入union
    hongmen  = UnionSet(n+1)#+1是建一个0到n+1的，0可以不用
    for _ in range(m):
        x,y = map(int,input().split())
        hongmen.join(x,y)
    start,end = map(int,input().split())
    return 1 if hongmen.is_union(start,end) else 0

if __name__ == "__main__":
    print(main())
    
    
```
