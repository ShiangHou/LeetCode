这题就是全排列中出现了重复的，需要去重

去重的逻辑和非递减子序列一样，准备一个set

当我的j位置没有出现在i位置的时候，才会去做交换，否则就不做

比如 1 1 2

1位置的1 如果没有在 0位置的1出现过才会去加，这里我们发现 0位置本来就是1 ，所以1位置的1就不会去加

当交换0 0 的时候，我们就会把自己加上去



```python 
    def permuteUnique(self, nums: List[int]) -> List[List[int]]:
        self.ans = []
        self.f(0,nums)
        return self.ans
    def f(self,i,nums):
        if i == len(nums):
            self.ans.append(nums.copy())
        
        else:
            used = set()#每次i位置要新建一个
            j = i
            while j < len(nums):
                if nums[j] not in used:#如果j位置的数字没有来过i位置
                    used.add(nums[j])
                    nums[i],nums[j] = nums[j], nums[i]
                    self.f(i+1,nums)
                    nums[i],nums[j] = nums[j], nums[i]
                j += 1
```
