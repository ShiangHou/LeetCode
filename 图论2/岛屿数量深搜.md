## 初步思路

这道题目的思路是这样

首先如果我到达了一个陆地，那么我就需要把这块陆地连在一起的所有的陆地都找到，然后标记出visited 

然后再往下搜下一个地方，如果这个地方被visited过了，那就继续

如果发现了还有新的，，那么就记录一下+1 ans

找到一块地方，去找连带着所有的陆地，就需要深搜

```python 
n,m = map(int,input().split())

#先建图
graph = []
for i in range(n):
    graph.append(list(map(int,input().split())))

#然后做一个dictrion，表示方向
dir = [[0,1],[1,0],[0,-1],[-1,0]]


#准备一个visted
visted = [[False]*m for _ in range(n)]

#深搜,对于一个陆地，找到他所有的相连的陆地，然后在visit里面标记
def dfs(graph,visted,i,j):
    if graph[i][j] == 0 or visted[i][j] == True:#遇到了是0的，或者已经访问过了，直接返回
        return 
    else:
        visted[i][j] = True#先把当前的陆地标记是True
        for next_x,next_y in dir:
            next_i,next_j = i+next_x,j+next_y
            if next_i>n-1 or next_i<0 or next_j>m-1 or next_j<0:
                #出界了
                continue
            else:#没出界
                dfs(graph,visted,next_i,next_j)
ans = 0
def main():
    #主程序是遍历所有的陆地
    ans = 0
    for i in range(n):
        for j in range(m):
            if graph[i][j] == 0 or visted[i][j] == True:
                continue
            else:#当前为1，并且还是false
                ans+=1
                dfs(graph,visted,i,j)
    return ans 

if __name__ == "__main__":
    print(main())
```
