## 入度 in-degree

in-degree的意思就是说，一个node如果被1个箭头指向了，说明这个node的in-degre就是1，有两个就是2

嘿嘿嘿

首先根节点肯定没有人被in-degree的

然后其他节点，正常情况加，要么是0个，要么是1个

如果发现有两个in degree，说明出事了，有一个node被俩人指了

所以问题转化为

每次建union的时候，统计一下所有的in—degree

此时有两种坏的情况

一种是，有人的in-degree是2，说明被俩人指了，那么需要斩杀的边一定是在这两条中的一条

那么其实此时随便斩杀一条都行，因为随便斩杀一个都可以符合条件，这里可以先加，然后判断，如果是情况一的话，直接把刚才刚加入的return就行

情况二是，所有的in-degree都是1！说明我们的根节点被别人指了！此时，我们可以知道，刚加入的那个边指向了0，导致所有的in-degree都是1，所以直接站杀掉

额，那情况1和情况2的操作可以合并啊，就是这样

把n加入并查集

然后加边，加一条统计一次所有人的in-degree

如果出现有人的是2，or 全都是1，立即把刚加的边return就行，

否则的话就正常加边就行

怎么统计in-degree？弄一个数组就行了，下标是node，元素是in-degree就可以


```python 
#先写一个并查集
class UnionSet:
    def __init__(self,n):
        self.father = [i for i in range(n)]
    def find(self,x):
        if self.father[x] == x:
            return x
        else:
            self.father[x] = self.find(self.father[x])
            return self.father[x]
    def union(self,x,y):
        father_x = self.find(x)
        father_y = self.find(y)
        if father_x != father_y:
            self.father[father_x] = father_y
    def is_same(self,x,y):
        return self.find(x) == self.find(y)


def main():
    n = int(input())
    in_degree = [0]*(n+1)#入度,这里只统计n+1把
    parent = UnionSet(n+1)

    for _ in range(n):
        x,y = map(int,input().split())
        #先看一下是不是一个老大，不是的话连图都没有加入

        #统计入度，每次清空
        #count_0 = 0#统计入度1的次数，好像不需要，只要看1是不是和n一样大，以及有没有出现2
        count_1 = 0#统计入度0的次数


        if not parent.is_same(x,y):
            #不是一个老大，直接存
            parent.union(x,y)
            #加一下入度
            in_degree[y] += 1
            #算一下入度
            for i in range(1,n+1):
                if in_degree[i] == 2:#出现一个2,说明刚才的x和y冲突了
                    ans_x,ans_y = x,y
                elif in_degree[i] == 1:
                    count_1 += 1
                #0不用管
            if count_1 == n:#即全都是1，说明刚加的这一个也有问题
                ans_x,ans_y = x,y
        else:#是一个老大了，更要直接斩杀
            ans_x,ans_y = x,y
    return ans_x,ans_y

if __name__ == "__main__":
    ans_x,ans_y = main()
    print(ans_x,ans_y)

```

这样写是不对的，因为不能当场斩杀，比如[[1,2], [2,3], [3,1], [4,1]]

你把 [4,1] 删了之后，剩下的图是 [[1,2], [2,3], [3,1]]……这依然是个环，根本不是树！

## 最终思路

思路应该是，先不要构建并查集，先看入度

就是说，我们先把edge给存起来，统一一下入度

然后扫一眼入度，如果发现有一个是2，说明进入了情况1

此时我们记住这两条边

建立一个新的并查集，先斩杀掉一条边，然后往并查集里面加东西，加之前看看是不是is_same

如果发现了is_same!,说明这就是一条环了，直接斩杀掉**我们在这两条边中选中的一条**

如果发现没环，那么直接返回另一条，说明另一条是罪魁祸首


如果入度没有2，那么情况其实就是冗余链接1，即就是一个环了已经

直接做加加加，发现有环，返回环


```python 
#先写一个并查集
class UnionSet:
    def __init__(self,n):
        self.father = [i for i in range(n)]
    def find(self,x):
        if self.father[x] == x:
            return x
        else:
            self.father[x] = self.find(self.father[x])
            return self.father[x]
    def union(self,x,y):
        father_x = self.find(x)
        father_y = self.find(y)
        if father_x != father_y:
            self.father[father_x] = father_y
    def is_same(self,x,y):
        return self.find(x) == self.find(y)

def has_2(edges,edge_2,n):#出现2了
    a,b = edge_2[0],edge_2[1]#两位候选人，先放弃b，先把a弄进去把

    s_2 = UnionSet(n+1)

    for edge in edges:
        if edge == b:#先不加b
            continue
        x,y = edge[0],edge[1]
        if s_2.is_same(x,y):#如果发现已经是一组的了，说明已经是环了，那么a就是需要斩杀的边
            return a[0],a[1]
        else:
            s_2.union(x,y)
    #没触发上面的，说明a不是，b才是罪魁祸首
    return b[0],b[1]

def all_1(edges,n):
    s_1 = UnionSet(n+1)
    for edge in edges:
        x,y = edge[0],edge[1]
        if s_1.is_same(x,y):#如果发现已经是一组的了，直接斩杀
            return x,y
        else:
            s_1.union(x,y)


def main():
    n = int(input())
    in_degree = [0]*(n+1)#入度,这里只统计n+1把
    edge = []
    for _ in range(n):
        edge.append(list(map(int,input().split())))

    #先把所有的边都找到，统计入度情况
    for i in range(n):
        in_degree[edge[i][1]] += 1
    
    #看下入度是不是有2或者全都是1或者0
    for i in range(1,n+1):
        if in_degree[i] == 2:#出现2了
            #把两条边找到
            edge_2 = []
            for k in range(n):
                if edge[k][1] == i:#如果指向的是i
                    edge_2.append(edge[k])
            
            #新写一个函数处理这种情况,直接return答案
            return has_2(edge,edge_2,n)
    #没找到出现的2,说明都是1，也是新写一个函数
    return all_1(edge,n)


    

if __name__ == "__main__":
    ans_x,ans_y = main()
    print(ans_x,ans_y)


    
```
