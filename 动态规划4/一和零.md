# 一和零

力扣474

## 初步思路

题目问，给了一个数组，数组里面有很多的字符串，每个字符串只由0和1构成，然后给了我们两个数m和n，问找到一个最大的子集，该子集中最多 有 m 个 0 和 n 个 1 


嘶，首先转化为背包问题，无非就是说，我从这个数组里去拿东西，最多能拿多少（在不撑爆的情况下）

我们dp定义的依旧是ij，

嘶，题目说的是最多有m和n个，那么也可以小于等于m和n

这里会有四个变量，i，j，遍历到了数组i，以及集合的最大长度，那么要怎么设置dp？

## 背包思路

厚米提示

定义一个二维数组 dp[i][j]，其严格的物理含义为：在容量上限为 i 个 0 和 j 个 1 的约束条件下，能够装入的最多字符串数量（即最大子集长度）。$i$ 的取值范围：$0 \le i \le m$$j$ 的取值范围：$0 \le j \le n$3. 

决策转移的思考方向

当遍历到数组中的某一个字符串时，假设经过统计，该字符串包含 zeros 个 0 和 ones 个 1。若要决定是否将该字符串加入当前集合，必然会消耗掉 zeros 和 ones 的容量，并使总价值增加 1。


最原始的三维的应该是

$$dp[k][i][j] = \max(dp[k-1][i][j], dp[k-1][i - zeros][j - ones] + 1)$$

当我们转成2维的时候，需要注意的是，其实也是从后往前填的，因为需要防止i是超的

```python 
class Solution:
    def findMaxForm(self, strs: List[str], m: int, n: int) -> int:
        dp = [[0]*(n+1) for _ in range(m+1)] 
        #dp[0][0] = 1#即所谓的空集合，组成0个0和0个1，不用，因为问的是有多少个字符串，空是什么都没装，所以是0，
        for str in strs:
            #先统计当前有几个0和几个1
            temp_0,temp_1 = 0,0
            for num in str:
                if num == "0":
                    temp_0 +=1
                else:
                    temp_1 += 1
            #然后开始填dp，注意是从后往前填，
            for i in range(m,temp_0-1,-1):
                for j in range(n,temp_1-1,-1):
                    #用当前的和不用当前的
                    dp[i][j] = max(dp[i][j],dp[i-temp_0][j-temp_1]+1)
        return dp[m][n]



```
