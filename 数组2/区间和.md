## 前缀和

超时的原因是我们每次都要遍历一边数组，非常的复杂，能不能取一下呢，因为我们要找一个区间的值，现在假设有一个数组，i位置存了0-i位置的数字的和，那么现在我要找a-b的和，是不是只需要拿0-b的所有和 减去 0-a的所有的和就行了。所以把这个前缀的数组构建好后直接做就行

```python 
import sys

input_data = sys.stdin.read()  #这个的意思就是说，将所有的输入都读到内存中

data = input_data.split()  #切分后，此时的data就是一个扁平的列表，里面是字符串

index = 0  #这个是用来读取data的下标的

n = int(data[index])
index += 1

nums = []
pre_sum = []
temp = 0
for i in range(n):
    nums.append(int(data[index + i]))
    temp += int(data[index + i])
    pre_sum.append(temp)



index += n

while index < len(data):#只要index不大于，说明就还有输入，一直做就行
    a = int(data[index])
    b = int(data[index+1])
    print(pre_sum[b]-pre_sum[a]+nums[a])#记得加上a
    temp = 0#重置
    index+=2

```
