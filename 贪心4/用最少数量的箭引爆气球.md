## 答案思路

道理是一样的，但是代码精简很多

首先，需要看一下两个气球是不是挨着的，如果没挨着（有重叠），那么就count+1就行

如果挨着了，我们更新一下挨着的新的边界（即挨着的俩气球的最小的右边界）当作新边界，即把这个两个看作一个整体就行
所以这样代码就好写了，如果说挨着，就更新当前气球（i+1）的右边界到 i和i+1的最小边界就行

```python 
    def findMinArrowShots(self, points: List[List[int]]) -> int:
        points.sort(key = lambda x: x[0])
        count = 1
        for i in range(len(points)-1):
            if points[i][1] < points[i+1][0]:#没挨着
                count += 1
            else:#挨着了
                points[i+1][1] = min(points[i][1],points[i+1][1])#更新新的右边界
        return count
```
