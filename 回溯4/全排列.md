## 答案

这道题核心思想就是说，我需要把所有的数都放在每个位置试一下，试玩后清理现场

比如数组是 x y z

让我的每个位置都到0去试一下，然后递归


这里还是要把全部的递归都想像成一颗二叉树

对于最初的 1 2 3
首先i是0 ，在内循环里面j是从0开始往len去递归

i = 0 这一层

for 循环，j从0 到len
    j = 0，
        0和0交换，还是 1 2 3，搜集到了1 2 3，

        递归f i+1，来到了i = 1，同时j也是1开始，我们先不管这一层搜集了啥，

        返回，交换回来，现场应该还是1 2 3
    j = 1
        0 1交换，所以说是此时数组是 2 1 3，搜集到了 2 1 3
        
        递归 f(i+1) 此时递归的数组是 2 1 3

        返回，交换回来，0和1，现场应该还是1 2 3
    
    j = 2 
        0 2交换，所以此时数组是3 2 1 搜集到了3 2 1

        递归f(i+1) 此时递归的数组是 3 2 1
        
        返回，交换回来，现场还是 1 2 3


上述的三个是二叉树的第一层（即三个分支），对于每个分支，依旧是递归，比如说


    j = 0，
        0和0交换，还是 1 2 3，搜集到了1 2 3，

        递归f i+1，来到了i = 1，同时j也是1开始，我们先不管这一层搜集了啥，

       然后这个i是1，j也是1，在这一层上有两个分支，分别是
            j = 1，然后1 1 交换，
            j = 2 然后2 1交换

```python 
    def permute(self, nums: List[int]) -> List[List[int]]:
        self.ans = []
        self.f(0,nums)
        return self.ans
    def f(self,i,nums):
        if i == len(nums):
            self.ans.append(nums.copy())
        else:
            j = i
            while j < len(nums):
                nums[i], nums[j] = nums[j],nums[i]
                self.f(i+1,nums)
                nums[i], nums[j] = nums[j],nums[i]
                j += 1


```
