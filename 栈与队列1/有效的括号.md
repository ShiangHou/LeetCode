# 有效的括号
力扣20

## 初步思路
看下题目，()[]{}是可以的，([])也可以，([)] 就不行，说白了就是，左括号无所谓，当出现右括号的时候，往后找第一个左括号一定要和这个右括号相匹配

所以思路就很显然，首先开始遍历，遇到左括号就压到栈里面

当遇到右括号的时候，从栈里面弹出看下是不是匹配，不匹配的直接False，匹配的话就把继续下一个（左括号也要弹出）

```python 
def isValid(self, s: str) -> bool:
    stack = []
    for i in s:
        if i == '(' or i == '{' or i == '[':
            stack.append(i)
        else:
            if not stack:#如果说遇到右括号，但栈里面还是空的，直接false
                return False
            else:
                if i == ')':
                    temp = stack.pop()
                    if temp != '(':
                        return False
                if i == '}':
                    temp = stack.pop()
                    if temp != '{':
                        return False
                if i == ']':
                    temp = stack.pop()
                    if temp != '[':
                        return False
    return False if stack else True#最后要保证栈全都被用完，否则也是false

```
