# 环形链表2
力扣142 非常经典的题目 

## 初步思路
依旧是靠着记忆在回想

首先肯定有个快慢指针，当快慢指针重合的时候，让慢指针回到原点，然后快慢指针再一起走，最后重合的就是入环时候的答案

由于忘记了第一次快慢指针重合是什么时候，分析一下开始快慢指针差多少，如果差一个，简单画一个可以发现可能永远都追不上，差两个也追不上

所以肯定不是差的问题，应该是速度，即快指针一下子走两步，慢指针走1步就行

```python 
def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if head == None:
            return None
        
        fast,slow = head,head
        while fast and slow:
            if fast.next == None or slow.next == None:
                #说明无环，直接返回
                return None
            fast = fast.next.next
            slow = slow.next
            if fast == slow:
                break
        slow = head
        while fast and slow:
            if fast == slow:
                return fast
            fast = fast.next
            slow = slow.next

```
