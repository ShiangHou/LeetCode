# 最长公共子序列

力扣1143

## 初步思路
构建一个二维的dp，仿照我们之前做的最长递增子序列的过程，定义为 以i和j结尾的，两个子串的最长公共子序列

递推公式是，如果i和j是一样的，那么说明出现了一个公共的字符，状态直接从i-1，j-1然后+1过来就行

如果说不一样，说明当前的不能延续这个最长公共子序列的长度，那么就要找之前的长度继承过来，怎么继承呢


继承方法是，放弃当前的i，用i-1之前的和j去匹配，那么也可以放弃j，用i和j-1区和i匹配

```python 
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        n = len(text1)
        m = len(text2)
        dp = [[0]*(m+1) for _ in range(n+1)]
        
        for i in range(1,n+1):
            for j in range(1,m+1):
                if text1[i-1] == text2[j-1]:
                    dp[i][j] = dp[i-1][j-1]+1
                else:
                    dp[i][j] = max(dp[i][j-1],dp[i-1][j])

                # ans = max(dp[i][j],ans)
        return dp[n][m]
```
注意初始化是0，因为可能没有公共的字符
