## 初步思路
very经典的题目，三个变量，四个步骤搞定

首先准备pre，_next,cur三个节点
_next  = cur.next 这一步是先存一下下一个节点
cur.next = pre #把当前的这个指向之前的
pre = cur 移动pre
cur = _next 移动cur

```python 
def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        pre = None
        _next = None
        cur = head
        while cur:
            _next = cur.next
            cur.next = pre
            pre = cur
            cur = _next
        return pre

```
