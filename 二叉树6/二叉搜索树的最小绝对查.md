# 二叉搜索树的最小绝对差
力扣530

## 初步思路
首先，这是一课二叉搜索树，题目要求说是返回任意节点的差

所以说，只去比较两个node和之前的其他的都是不行的，最小的差值可能会跨层出现，所以最好的方式是，往下递归的时候带着上下限去递归，然后去计算就行

梳理一下思路

首先写一个递归函数，这个函数应该是要返回最小值的

入参是node，min，max。对于一个node来说，如果他是右节点，那么它至少是要比父节点大，且至多是要比max的小

换句话说，我们要找node的最窄的区间范围，这个范围就是

举个例子
A

B C

D E F G

有
B < A,  C > A
D < B, B < E < A

所以在比较的时候这样比就行，遍历到搜索二叉树的每一个节点就行

## 最终答案

要知道：二叉搜索树的中序遍历就是严格升序的



那么对于一个升序的数组，直接算两个数之间的差就行

```python 
    def getMinimumDifference(self, root: Optional[TreeNode]) -> int:
        tree_list = self.findtree(root,[])
        ans = float('inf')
        for i in range(len(tree_list)-1):
            ans = min(ans,tree_list[i+1]-tree_list[i])
        return ans

    def findtree(self,node,ans):
        if not node:
            return
        self.findtree(node.left,ans)
        ans.append(node.val)
        self.findtree(node.right,ans)
        return ans
```
