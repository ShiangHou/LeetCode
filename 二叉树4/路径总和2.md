## 初步思路

跟路径总和1是一样的，只不过这次要记录

在路径总和1的时候我当时用了全局变量，这次就可以用了

直接递归里面一个path一个target，到最后判断是的话就直接存到最终的ans里面就行哈哈哈

```python 
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> List[List[int]]:
        if not root:
            return []
        self.ans = []
        self.f(root,[],targetSum)
        return self.ans   

    def f(self,node,path,target):
        if not node.left and not node.right:
            if target == node.val:
                self.ans.append(path.append(node.val))
        if node.left:
            self.f(node.left, path.append(node.val),target-node.val)
        if node.right:
            self.f(node.right,path.append(node.val),target-node.val)

```

这样写是有bug的，因为path.append(node.val) 本身是一个操作，不是一个值，他的值就是None
