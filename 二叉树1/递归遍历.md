## 初步思路 

一个二叉树的数据结构长这样
```python 
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
```
递归遍历之前也是very的基础

首先函数整体的入参是一个二叉树节点node

首先先看node是不是空的，如果是空的直接返回空就行

如果不是空的，按照要求的遍历顺序直接写就行，比如说前序遍历就是左中右，中序遍历就是中左右，后续遍历就行右中左就行

说错了，前序遍历是中左右(力扣144)，后序遍历是左右中，中序遍历是左中右,

记忆技巧是，由于中的value，所以如果是前序，就要先value，所以是中左右，后序是最后value，所以是


前序遍历
```python 
def preorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
    ans = []
    def Traver(node):
        if node == None:
            return 
        ans.append(node.val)
        Traver(node.left)
        Traver(node.right)
    Traver(root)
    return ans
```
后序遍历

```python 
def postorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
    ans = []
    def Traver(node):
        if node == None:
            return 
        Traver(node.left)
        Traver(node.right)
        ans.append(node.val)
    Traver(root)
    return ans

```
