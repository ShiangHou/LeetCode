## 初步思路

这题其实是一样的，就是说要找到所有的孤岛的面积

但是这里的孤岛定义不一样，所谓的孤岛，必须要是前后左右都不和“边缘”链接

换句话说，这个孤岛必须是周围全是0的水，

所以说，我们在dfs的时候，直接加一些判定条件，

怎么加呢，emmmm，首先还是先用一个temp来去存放这个岛屿的面积就行，如果发现越界了，return一个flag是False，表示这个弃用就行

```python 
n,m = map(int,input().split())

#建图
graph = []
for i in range(n):
    graph.append(list(map(int,input().split())))

#方向
direction = [[1,0],[-1,0],[0,1],[0,-1]]

visited = [[False]*m for _ in range(n)]

#表示这个岛的数量能不能用
flag = True

def dfs(graph,i,j,visited):
    global flag
    ans = 1
    if graph[i][j] == 0 or visited[i][j] == True:
        return 0
    visited[i][j] = True
    for x,y in direction:
        next_i,next_j = i+x,j+y
        if next_i<0 or next_i >n-1 or next_j<0 or next_j >m-1:
            #如果越界了，记录是False
            flag = False
            continue
        elif graph[next_i][next_j] == 1 and visited[next_i][next_j] == False:
            ans += dfs(graph,next_i,next_j,visited)
    return ans 
    


#主循环
def main():
    global flag #python 中，如果函数里面有 = 的赋值操作，就会认为是局部变量，这里声明一下
    ans = 0
    for i in range(n):
        for j in range(m):
            #应该是先运行一遍
            if graph[i][j] == 1 and visited[i][j] == False:
                temp_ans = dfs(graph,i,j,visited)
                #得到一个临时答案，需要判断要不要加进入
                if flag == True:
                    ans+=temp_ans
                else:#如果发现是False，说明这片陆地不能用，找下一片，重新找的时候重制一下flag
                    flag = True


    return ans


if __name__ == "__main__":
    print(main())
            

```
