## 初步思路

这个very的简单啊，就是之前，我们不是深搜或者广搜，找到的该陆地以及该陆地连接的陆地，我们在每一次调用的记录一下数量（即在dfs或者bfs函数里面，因为这俩都是找该陆地连接的所有陆地）

用dfs写吧，递归还是要多练

```python 
n,m  = map(int,input().split())

#建图
graph = []
for i in range(n):
    graph.append(list(map(int,input().split())))

#方向
direction = [[1,0],[-1,0],[0,1],[0,-1]]

visited = [[False]*m for _ in range(n)]

#深搜，负责找到i j连接的所有的陆地，返回大小
def dfs(graph,i,j,visited):
    if graph[i][j] == 0 or visited[i][j] == True:
        #如果到头了，或者不是这个，说明没有水，直接返回0
        return 0
    #把当前标记为True
    visited[i][j] = True
    #自己的面积是1
    count = 1
    #开始深搜
    for x,y in direction:
        next_i,next_j = i+x,j+y#标记下一个位置
        if next_i<0 or next_i>n-1 or next_j<0 or next_j>m-1:
            continue
        elif graph[next_i][next_j] == 1 and visited[next_i][next_j] == False:
            count+=dfs(graph,next_i,next_j,visited)#每次把返回的搜集起来
    #最后return 一下count
    return count

def main():
    ans = 0
    for i in range(n):
        for j in range(m):
            if graph[i][j] == 1 and visited[i][j] == False:
                ans = max(dfs(graph,i,j,visited),ans)
    return ans

if __name__ == "__main__":
    print(main())

```

这里注意一下写法，即这个递归中的传递count的，需要把上下左右四个方向的深搜递归结果的count返回过去，加在一起，在返回给上级
