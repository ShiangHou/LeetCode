# 最长回文子序列

## 初步思路

这次问的是最长回文子序列，子序列就是可以从中间删除，

这里问的是最长的，那么我们依旧是考虑刚才的，以i和j（包含）的是不是回文的，然后这里我们需要判断了，就是递推公式需要变一下

当左右相等的时候，皆大欢序，继续传递，dp[i+1][j-1]

当左右不等的时候，我们需要看一下缩进去里面哪个是的，即dp[i+1][j] or dp[i][j-1]

哎不对，这里问的是长度，不是个数，emmmm

那是不是可以从中间就开始，然后不断往外扩，应该是不行的

经过厚米的提醒，这个做法是对的，但是我们的定义是，i开头j结尾的最大回文子序列的长度

那么左右相等的时候，直接等于dp[i+1][j-1]+2就行

然后左右不等的时候，直接max(dp[i+1][j] , dp[i][j-1])

然后遍历顺序应该是一样的，

初始化的时候，dp[i][0]的就是任意开头，然后0结尾，00开头是1，其他的都是0吧

0j开头的话，emmm也是需要去判断，算了先都初始化为0，除了对角线上是1，其他都是0

```python 
class Solution:
    def longestPalindromeSubseq(self, s: str) -> int:
        n = len(s)
        dp = [[0]*n for _ in range(n)]
        for i in range(n-1,-1,-1):
            for j in range(i,n):
                if i == j:#先处理一样的
                    dp[i][j] = 1
                elif s[i] == s[j]:
                    dp[i][j] = dp[i+1][j-1]+2
                else:
                    dp[i][j] = max(dp[i+1][j],dp[i][j-1])
        return dp[0][n-1]
```
