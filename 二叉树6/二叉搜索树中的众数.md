## 初步思路
依旧是very的简单啊，我把数都统计一遍存在一个dict里面，然后找最大的不久行了？


梳理一遍思路，首先遍历出来数组，然后存入hash，然后遍历第一遍hash去找到最大的众数出现了几次，然后再遍历一遍把所有的出现的这个次数的获取就行

```python 
def findMode(self, root: Optional[TreeNode]) -> List[int]:
        ans = []
        def find(node):
            if not node:
                return 
            find(node.left)
            ans.append(node.val)
            find(node.right)
        find(root)
        counts = {}
        for i in ans:
            counts[i] = counts.get(i, 0) + 1#注意这里就是，get去拿值，如果拿不到就返回默认值0
        ans_count = float('-inf')
        final_ans = []
        for num,count in counts.items():
            ans_count = max(ans_count,count)
        for num,count in counts.items():
            if count == ans_count:
                final_ans.append(num)
        return final_ans

```


## 进阶

要求我们不再用额外的空间了，即在树上遍历

回想一下二叉搜索树的情况，当我们中序遍历的时候，是一个递增的（应该是不递减的），

如果是一个数组就好办了，我们用两个指针，一个指针留在当地，如果是相等的，一个指针往后走，直到不相等，此时统计一下数量就行

但是还有一个问题，怎么处理重复的众数呢

答案很清晰，如果一个新的count突破了记录，那么就删除之前所有的，只留它，如果等于的化就把他加进去，小于的化就不管了

此外，这个操作要在树上就走完，不能树转成列表再弄

这里其实可以就维护几个全局变量就行，递归只用来去遍历树


```python 
    def findMode(self, root: Optional[TreeNode]) -> List[int]:
        self.ans = []
        self.cur_count = 0
        self.max_count = 0
        self.pre_node = None
        self.tree(root)
        return self.ans


    def tree(self,node):
        if not node:
            return 
        self.tree(node.left)
        if self.pre_node == None:#这是开始的第一个节点
            self.cur_count = 1
        elif self.pre_node.val == node.val:#当前节点和自己是一样的
            self.cur_count += 1 
        
        else:#不一样
            self.cur_count = 1


        self.pre_node = node#都弄好后，换一下pre节点 

        if self.cur_count > self.max_count:#破纪录了
            self.max_count = self.cur_count
            self.ans = [node.val]
        elif self.cur_count == self.max_count:#注意这里是elif，要是互斥的判断
            self.ans.append(node.val)
            
 
        self.tree(node.right)



```
