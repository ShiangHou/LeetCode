## 思路

正好左神讲到了就听了一下思路

首先我们可以获取p和q的指，假设p时30，q是100

我们做如下操作

1）如果遍历到node是大于q的，那么往左走

2）如果遍历node是小于p的，往右走

3）如果遍历到p或者q，那么说明就是交汇处，直接返回q或者p

3）如果遍历到一个节点是p< node < q,说明是交汇处，直接返回

解释一下，1和2好理解，因为是二叉搜索树，如果当前节点太大的化，说明都在左子树上，太小的化都在右子树上

解释一下3，首先当我1和2都没遇到，走着走到了p处，那么此时，那个小的一定是在p的子树上，所以此时p就是公共

同理，4很好理解，上述3个情况都没发生，走到一个节点是p< node < q，那么说明左边是p，右边是q啊，那么该节点就是第一次分叉，也是第一次汇集的点

def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
    #一直让p是小的那个，q是大的那个
    if q.val < p.val:
        p,q = q,p
    if not root or root == p or root == q:
        return root
    if root.val < p.val:
        return self.lowestCommonAncestor(root.right,p,q)
    elif root.val > q.val:
        return self.lowestCommonAncestor(root.left,p,q)
    elif p.val < root.val < q.val:
        return root
