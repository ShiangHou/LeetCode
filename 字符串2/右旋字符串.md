# 右旋字符串
卡码网https://kamacoder.com/problempage.php?pid=1065

## 初步思路
这里的题目的意思就是说，将一个字符的后k个移到前面去，比如说abcdefg，k=2,那么移动后就是fgabcde

初步的想法是，首先将后k位置切分开，单独拿出来，然后把后面被切的拼上就行

所以问题在于怎么去做这些，字符串可以按照[:]这样操作嘛，Gemini告诉我是可以的

注意，这个[:]也是左闭右开

```python 
k = int(input())
s = str(input())
n = len(s)
part1 = s[:n-k]
part2 = s[n-k:]
print(   part2+part1  )

```
