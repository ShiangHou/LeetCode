# 先序序列化和反序列化

首先说明一下什么是序列化和反序列化,说白了，就是把一颗二叉树拍扁成一个序列（比如列表等），反序列化就是把一个序列转成二叉树就行

为什么要序列化，其实一颗二叉树是存在内存里面的，如果可以变成一个序列，比如字符串，那么我就可以弄到硬盘里面去储存

注意序列化的时候要把所有的none都加进去

先序的可以是唯一的，后序的也是，但是中序的不能序列化，因为不唯一

```python 
class Codec:

    def serialize(self, root):
        res = []
        def dfs(node):
            if not node:
                res.append("#")
                return 
            res.append(str(node.val))
            dfs(node.left)
            dfs(node.right)
        dfs(root)
        return ",".join(res)
        

    def deserialize(self, data):
        vals = iter(data.split(","))
        def build():
            val = next(vals)
            if val == "#":
                return None
            node = TreeNode(int(val))
            node.left = build()
            node.right = build()
            return node
        return build()
```
