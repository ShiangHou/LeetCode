# 最佳买卖股票含冷冻期
力扣309

## 初步思路

这题就是有了所谓的冷冻期

看了一下题目的例子，所谓的冷冻期就是说，我卖出后必须要等一天才能买入

买入后第二天可以卖出

那么这个其实就是把递推方程改一下就行了，看了一下，这里买卖的次数可以无限次，那就继续两个状态就行了
依旧是dp[i][j] 其中i是n，j是0和1

初始化，dp[0][0]还是0
dp[0][1]还是-price[i]

递推过程中，

dp[i][0] 依旧是两种，一种是i-1也是没持有，即 dp[i-1][0] 一种是之前卖了，这里买卖之间是可以没有冷却期的，所以就是dp[i-1][1]+price[i],最后就是max(dp[i-1][0],dp[i-1][1])

dp[i][1]是持有，这个持有的话，一种是之前就是持有，即dp[i-1][1]，另一种是之前没持有，

嘶，这里要区分一下冷却期吧，算了加一个状态2当作冷却期

那么由于有冷却期，所以就是冷却期的状态，即dp[i-1][2] -price[i]

加了冷却期，那么冷却期其实只有一种状态,就是从刚卖的继承

dp[i][2] = dp[i-1][0]

好像也不是，算了不弄2状态了，

因为买的时候的第二种情况，即之前刚卖，这个之前必须要是前一天的，所以就是dp[i-2][0]才行，这样就可以了

那么我初始的时候需要再初始状态1才行，即
dp[1][0] = max(dp[0][0],dp[0][1]+prices[1])

这个1 1就是第二天买了，就这一种情况，所以就是
dp[1][1] = -price[1]

这样就初始化好了

```python 
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        n = len(prices)
        if n == 1:
            return 0
        dp = [[float('-inf')]*2 for _ in range(n)]
        dp[0][0] = 0
        dp[0][1] = -prices[0]
        dp[1][0] =max(0,prices[1]-prices[0])
        dp[1][1] = max(-prices[0],-prices[1])

        for i in range(2,n):
            dp[i][0] = max(dp[i-1][0],dp[i-1][1]+prices[i])
            dp[i][1] = max(dp[i-1][1],dp[i-2][0]-prices[i])
        return dp[n-1][0]

```

注意，这个dp[1][1] = max(-prices[0],-prices[1]) 初始化应该如此，因为是要选一个成本小的就行
