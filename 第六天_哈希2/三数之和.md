#三数之和

力扣15
## 初步思路
问的是一个数组里面存不存在三个不同的数（下标不同）使他们加一起等于0

第一个想法依旧是n3复杂度疯狂暴力遍历哈哈哈哈哈哈

这题看了说明，说是要用双指针做，

## 答案思路

这题是要求返回三元组的数值，因此，我们可以先对数组排序，排序完后，对于每一个i，left是i+1，right是n-1，然后三个数开始算，如果大了，就移动right，小了，就移动left，遇到了就记录就行（遇到了还是要继续，因为有可能下面再-1+1也是的，遇到了left和right一起动就行）

```python 
 def threeSum(self, nums: List[int]) -> List[List[int]]:
        ans = []
        nums.sort()#记得先排序
        for i in range(len(nums)):
            if nums[i] > 0:#如果i都大于0了，后面无论怎么加都是无法等于0的
                break


            left = i+1
            right = len(nums)-1
            if i > 0 and nums[i] == nums[i-1]:#跳过i相通的
                continue
            while left < right:
                if nums[i] + nums[left] +nums[right] == 0:
                    ans.append([nums[i],nums[left],nums[right]])
                    #这里也需要去重，left和right
                    while left < right and nums[left] == nums[left+1]:
                        left+=1
                    while left < right and nums[right] == nums[right-1]:
                        right-=1
                    #left和right去重后记得往后加一下
                    left+=1
                    right-=1
                elif nums[i] + nums[left] +nums[right] >0:
                    right-=1
                else:
                    left += 1
        return ans

```
