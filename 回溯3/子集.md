## 初步思路
就是需要找到这个几何集合的所有子集就行，一个简单的递归

首先想想一个二叉树，对于1 要还是不要，然后不断往下递归就行，和之前写的那个组合感觉差不多,分为要i和不要i，由于这题是说所有的组合，空也算，所以终止条件就是遍历到头就行

```python 

def subsets(self, nums: List[int]) -> List[List[int]]:
        self.ans = []
        self.f(0,nums,[])
        return self.ans
    


    def f(self,i,nums,path):
        if i >=len(nums):
            self.ans.append(path.copy())
            return 
        #选i
        path.append(nums[i])
        self.f(i+1,nums,path)
        #弹出
        path.pop()

        #不要i
        self.f(i+1,nums,path)

```
