## 初步思路

图论要用ACM格式，因为需要自己建图，构建起始点和末尾点

主函数，题目说是要我们找到从起始点到结尾点的所有路径

这个其实就是递归啊，在图中，我需要遍历上下左右四个，如果有1的话，就加入path，往那边走，直到走到了end，然后记录，return，就行了

```python 
#先读取输入输出建图
n,m = map(int,input().split())
graph = [[0]*(n+1) for _ in range(n+1)]
for a in range(m):
    i,j = map(int,input().split())
    #需要记录第一个和最后一个，即终点和目的地
    if a == 0:
        start = i
    if a == m-1:
        end = j
    graph[i][j] = 1

ans = []#最终的答案存放

#写一个深搜，入参是起始点，终点，当前点，path
def dfs(start,end,cur,path):
    #递归的终止条件，即抵达了end
    if cur == end:
        path.append(cur)
        ans.append(path.copy())
        return 
    
    #上下左右开始遍历，找到有1的，然后往下走
    #注意，图的graph[i][j] = 1表示了从i到j有一条路
    #遍历所有的n,如果发现dp[cur][j] = 1，直接往里闯
    path.append(cur)#先把当前的加入
    for j in range(n+1):
        if graph[cur][j] == 1:
            #直接闯，加入的在后面加入
            dfs(start,end,j,path)
            #记得清理
            path.pop()
        else:
            continue


dfs(1,n,1,[])
if ans == []:
    print(-1)
for i in ans:
    print(" ".join(map(str,i)))


```
