# 划分字母区间
 力扣 763

## 初步思路
题目说，给我们一个字符串要求进行切分，

要求是，一个字母只能出现在一段里面，不能出现在两段。最后返回分割后的

首先最先想到的是，每一个字母出现的最后的下标

因为不能出现在别的，所以对对于一个字母来说，自己最后一次出现的就行可能的分割点

我们可以统计每个字母的最后的分割点就行

首先先遍历一遍，先存下来每一个字母的出现的最后的下标是多少

然后再遍历一遍思考

什么时候切割
假设是ba，a当前元素的下标，假设字典里存的是8，但是b在字典里面存的是11，那么其实还是要打到11，那么我们就需要去维护一个start，一个end，初始状态就是0

然后我们开始遍历，遍历到b的时候，记录下是11，因此end就是11，

遍历到a的时候，是8，那么其实8小于11，那么这个end是不变的，

最后，当我们真正的遍历到11的位置，并且如果end也是11的时候，这时就可以切了，即i追上了end，即“前面所有元素的最大扩展值”，记录一下end-start，然后继续，start end整理一下，就行。

如果走到最后了，也记录一下end-start，防止有的字符串就是整个在一起才可以

```python 
class Solution:
    def partitionLabels(self, s: str) -> List[int]:
        n = len(s)
        max_index= {} 
        for i in range(n):#一个是下标，一个是字母
            #这里直接不断的往后刷新就行了，因为肯定是越往后数字越大就行
            max_index[s[i]] = i

        start,end = 0,0
        ans = []
        for i in range(n):
            #先更新当前字母的最大下标
            end = max(end,max_index[s[i]])
            #如果说当前的i就是end，那么就切一刀
            if i == end:
                ans.append(end-start+1)
                start,end = i+1,i+1#更新start和end
            # #如果到头了，也砍一刀
            # if i == n-1:
            #     ans.append(end-start)
            
        return ans
```
注意，到头了不用砍，因为i自然会追到最后的
