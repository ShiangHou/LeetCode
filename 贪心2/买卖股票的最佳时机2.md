# 买卖股票的最佳时机2

力扣122 

这道题就是说，给了一组数组，是股票的价格，问怎么买卖使得利润最大

这题very的简单，首先利润的本质就是股票差

我只需要把后面的大于前面的所有的差值找到就行，即把握住所有升高的曲线

```python 
    def maxProfit(self, prices: List[int]) -> int:
        profit = 0
        for i in range(len(prices)-1):
            if prices[i+1] > prices[i]:
                profit += prices[i+1]-prices[i]
        return profit


```
