## 初步思路

题目说要找到两个word，使得两个字母修改后长度相同

看了一下例子，比如 eat和ate，答案是2 ，因为eat先变成at再变成ate

leetcode和etco，就是直接吧leetcode删除前俩和后俩就行

emmm，这个东西，看了一下，首先是，如果两个字符重复的部分其实不用变，需要变的是没重复的部分，
此外是word1和word2相同，假设word2的长度是3，重复的长度是2，word1的长度也是3，那么其实就是word2的长度减去重复的，加上word1的长度减去重复的就行

这个重复的是连续重复才行，如果不是连续重复，说明中间的还是要删除

思路就是，统计最长连续重复的，也就是最长重复子序列

然后最后输出刚才的公式就行

```python 
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m = len(word1)
        n = len(word2)
        dp = [[0]*(n+1) for _ in range(m+1)]
        max_len = 0
        for i in range(1,m+1):
            for j in range(1,n+1):
                if word1[i-1] == word2[j-1]:
                    dp[i][j] = dp[i-1][j-1]+1
                    max_len = max(max_len,dp[i][j])
                else:
                    dp[i][j] = 0
        return m+n-2*max_len
```
嘶，好像是不对，不应该强制是连续的，不连续也可以啊,中间去掉就行，所以应该求最长公共子序列

```python 
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m = len(word1)
        n = len(word2)
        dp = [[0]*(n+1) for _ in range(m+1)]
        max_len = 0
        for i in range(1,m+1):
            for j in range(1,n+1):
                if word1[i-1] == word2[j-1]:
                    dp[i][j] = dp[i-1][j-1]+1
                    max_len = max(max_len,dp[i][j])
                else:
                    dp[i][j] = dp[i-1][j]
```
