## 初步思路
跟打家劫舍一样，只不过说，首位两端是相连的


emmmm，这样吧，第一个如果打了，那么最后一个就不能打，直接按照我们刚才返回的dpn-1就行

如果第一个没打，即dp0和dp1都是0，那么代码不变，直接返回dpn

然后两个取最大就行


```python 
class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 1:
            return nums[0]#注意一下，这里如果长度是1会直接返回0
        dp1 = [0]*(n+1)

        #先处理0位置没打的,没打就是0就行
        for i in range(2,n+1):
            dp1[i] = max(dp1[i-1],dp1[i-2]+nums[i-1])
        
        #再看位置0打了，最后返回的是dpn-1
        dp2 = [0]*(n+1)
        dp2[1] = nums[0]
        for i in range(2,n+1):
            dp2[i] = max(dp2[i-1],dp2[i-2]+nums[i-1])

        return max(dp1[n],dp2[n-1])
```

子集 A： 允许的搜索区间为从第 $0$ 栋到第 $n-2$ 栋（从物理上彻底切除最后一栋房屋，规避首尾相连的风险）。

子集 B： 允许的搜索区间为从第 $1$ 栋到第 $n-1$ 栋（从物理上彻底切除第一栋房屋）。
