# 删除字符串中所有相邻重复项
力扣1047
## 初步思路
题目要求和消消乐差不多，abbaca，先消掉bb，变成aaca，再消掉aa，最后留ca

其实就是一个栈，最初进入a，然后进入b，用b和a比一下，不一样就加入，一样就弹出，然后下一个这样就是

a
ab
a
空
a
c
结束

```python 
 def removeDuplicates(self, s: str) -> str:
        stack = []
        stack.append(s[0])
        for i in s[1:]:
            if stack and i == stack[-1]:
                stack.pop()
            else:
                stack.append(i)
        return "".join(stack)

```
