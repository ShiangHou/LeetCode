## 初步思路

这题跟不同路径是一样的，只不过，给了我们一个数组，如果是1的话说明有那个石头拦住了过不了

还是只能向下和向右

那么这题其实还好的，我们在写dp的时候，如果发现已经是1了，那么就把它弄成0就行啊，这样别人调用他的时候就是0就行

记得在初始化的时候，如果遇到1了，那么也要弄成0，所以初始化的时候最好是也用dp依赖的形式

```python 
class Solution:
    def uniquePathsWithObstacles(self, obstacleGrid: List[List[int]]) -> int:
        m = len(obstacleGrid)
        n = len(obstacleGrid[0])

        dp = [[0]*n for _ in range(m)]
        dp[0][0] = 1 if obstacleGrid[0][0] == 0 else 0
        
        for i in range(1,m):
            if obstacleGrid[i][0] == 1:
                 dp[i][0] = 0
                 continue
            dp[i][0] = dp[i-1][0]
        
        for j in range(1,n):
            if obstacleGrid[0][j] == 1:
                 dp[0][j] = 0
                 continue
            dp[0][j] = dp[0][j-1]
        
        for i in range(1,m):
            for j in range(1,n):
                if obstacleGrid[i][j] == 1:
                    dp[i][j] = 0
                    continue
                dp[i][j] = dp[i-1][j] + dp[i][j-1]
        return dp[m-1][n-1]
```
